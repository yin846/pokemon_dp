CPS_SocGetEport

C Specification

#include <nitroWiFi/cps.h>
u16 CPS_SocGetEport (void);

Arguments

None.

Return Values

New ephemeral port number

Description

Gets the new ephemeral port number. Used for FTP applications, etc. Does not block.
