CPS_SocGets

C Specification

#include <nitroWiFi/cps.h>
u8 *CPS_SocGets (void);

Arguments

None.

Return Values

Input character

Description

Inputs one line. One line is a unit delimited by CR and LF. CR and LF are removed. This address must be set to linbuf of CPSSoc. This blocks.

See Also

CPS_SocGetChar, CPSSoc
