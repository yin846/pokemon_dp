CPS_SocConsume

C Specification

#include <nitroWiFi/cps.h>
void CPS_SocConsume (u32 len);

Arguments

len Receive buffer length

Return Values

None.

Description

Clears data from the receive buffer. See CPS_SocRead. May block while using TCP. Does not block while using UDP/PING.

See Also

CPS_SocRead
