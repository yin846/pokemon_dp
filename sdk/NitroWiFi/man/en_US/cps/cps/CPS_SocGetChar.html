CPS_SocGetChar

C Specification

#include <nitroWiFi/cps.h>
int CPS_SocGetChar (void);

Arguments

None.

Return Values

Input character.

Description

Waits for a single character entry, and then returns that character. If the connection ends, returns EOF. This blocks.
