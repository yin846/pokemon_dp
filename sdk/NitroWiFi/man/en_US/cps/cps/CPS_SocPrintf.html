CPS_SocPrintf

C Specification

#include <nitroWiFi/cps.h>
void CPS_SocPrintf(const char *format_str, ...);

Arguments

format_str Output format string (format character string).
... Variable number of arguments.

Return Values

None.

Description

Outputs a character string with format control. The buffer address must be set to outbuf of CPSSoc. Buffering continues until outbuf is full. Only %d, %u, %x, %X, %c, and %s are used for control. Width cannot be specified for %c and %s. This blocks.

See Also

CPS_SocPutChar, CPS_SocPuts, CPS_SocFlush, CPSSoc
