CPS_SslAddRandomSeed

C Specification

#include <nitroWiFi/ssl.h>
void CPS_SslAddRandomSeed(void *random_seed, u32 len);

Arguments

random_seed Random number pool.
len Random number pool size

Return Values

None.

Description

Sets the random number pool to be used by the SSL random number generator. The setting is set by adding entropy to the existing random number pool.
When SSL first needs a random number, it checks to see if a random number pool has been set. If not, it sets the random number pool internally. However, since this random number is not good for encryption purposes, CPS_SslAddRandomSeed() should be used to set the random number if at all possible.
This does not block.
This can be called even when the CPS library cannot be used.
