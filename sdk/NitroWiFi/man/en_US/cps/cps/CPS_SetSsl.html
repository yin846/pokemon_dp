CPS_SetSsl

C Specification

#include <nitroWiFi/cps.h>
void CPS_SetSsl(int mode);

Arguments

mode Parameter

Return Values

None.

Description

Passes a non-zero parameter to set to SSL mode. All subsequent TCP-related APIs are performed using SSL.
After this is set, if a different value is set before using CPS_SocRelease(), the operation is not guaranteed.
This blocks.
