CPS_CV_IPv4

C Specification

#include <nitroWiFi/cps.h>
#define CPS_CV_IPv4(ip) (u8)((ip) >> 24), (u8)((ip) >> 16), (u8)((ip) >> 8), (u8)(ip)

Arguments

ip IP address

Description

This enables u32 IPv4 addresses to be used as printf parameters in order to convert them to character string expressions.
