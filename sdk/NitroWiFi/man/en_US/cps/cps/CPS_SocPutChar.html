CPS_SocPutChar

C Specification

#include <nitroWiFi/cps.h>
void CPS_SocPutChar(char c);

Arguments

c Output character

Return Values

None.

Description

Outputs a single character. The buffer address must be set to outbuf of CPSSoc. Buffering continues until outbuf is full. This blocks.

See Also

CPS_SocPuts, CPS_SocPrintf, CPS_SocFlush, CPSSoc
