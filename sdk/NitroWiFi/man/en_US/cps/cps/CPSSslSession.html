CPSSslSession

Definitions

#include <nitroWiFi/cps.h>
typedef struct {
u8 sessionID [32];
u8 master_secret [48];
u32 when;
CPSInAddr ip;				// 0 when entry is for server
u16 port;					// 0 when entry is for server
u8 valid;					// non 0 if this entry is valid
u8 padding;
} CPSSslSession;

Description

This structure defines the SSL connection. CPSSslConnection.session points to this structure.
