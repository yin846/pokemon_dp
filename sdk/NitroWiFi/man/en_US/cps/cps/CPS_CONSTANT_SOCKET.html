CPS_STT_*

C Specification

#include <nitroWiFi/cps.h>
enum {
	CPS_STT_CLOSED = 0,
	CPS_STT_LISTEN,
	CPS_STT_SYN_SENT,
	CPS_STT_SYN_RCVD,
	CPS_STT_ESTABLISHED,
	CPS_STT_CLOSE_WAIT,
	CPS_STT_LAST_ACK,
	CPS_STT_FIN_WAIT1,
	CPS_STT_FIN_WAIT2,
	CPS_STT_CLOSING,
	CPS_STT_DATAGRAM,
	CPS_STT_PING
};

Description

Constants set in CPSSoc.state. Indicate the socket state. Users must not change these values.

CPS_STT_CLOSED
CPS_STT_LISTEN
CPS_STT_SYN_SENT
CPS_STT_SYN_RCVD
CPS_STT_ESTABLISHED
CPS_STT_CLOSE_WAIT
CPS_STT_LAST_ACK
CPS_STT_FIN_WAIT1
CPS_STT_FIN_WAIT2
CPS_STT_CLOSING		 Each element indicates the corresponding TCP states.
CPS_STT_DATAGRAM Indicates that the socket is in UDP mode.
CPS_STT_PING Indicates that the socket is in PING (ICMP echo) mode.
