SOC_AddRandomSeed

C Specification

#include <nitroWiFi/soc.h>
void SOC_AddRandomSeed ( void* seed, u32 len );

Arguments

seed A pointer to the seed for generating a pool of random numbers.
con The length of the seed specified by the seed pointer.

Return Values

None.

Description

Generates a pool of random numbers for generating the random numbers used in SSL communication. This function may be called prior to initializing the communication library.

Prior to commencing SSL communications, be certain to provide this function with sufficient entropy to ensure safe SSL communications. It is also recommended to subsequently provide fresh entropy on a regular basis.

For data specified by the seed pointer, sampling data from the microphone, for example, would afford a high amount of entropy, but it is impractical to constantly continue sampling. The OS_GetLowEntropyData is available as an economical and easy way to obtain a small entropy source. However, the data entropy obtained from calling the OS_GetLowEntropyData function once is not very high, so prior to SSL communication, call the OS_GetLowEntropyData function periodically and pass the data obtained to the SOC_AddRandomSeed function to provide sufficient entropy for generating random numbers.
