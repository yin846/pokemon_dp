SOC_GetHostID

C Specification

#include <nitroWiFi/soc.h>
long SOC_GetHostID( void );

Arguments

None.

Return Values

Host's primary IPv4 address (Network byte order)

Description

This function returns the host's primary IPv4 address.

Note:SOC_GetHostID() returns SOC_INADDR_ANY while DHCP host configuration is taking place.
