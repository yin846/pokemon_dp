SOC_InetPtoN

C Specification

#include <nitroWiFi/soc.h>
int SOC_InetPtoN ( int af, const char* src, void* dst );

Arguments

af Address family (must be SOC_PF_INET)
src Pointer to passed string. (Example: ddd.ddd.ddd.ddd)
dst Pointer to buffer where address in numeric format is recorded.

Return Values

0 Successful.
1 src is not in valid IPv4 address format.
SOC_EAFNOSUPPORT af is not SOC_PF_INET.

Description

This function converts IP addresses from dot-decimal notations to binary addresses.

See Also

SOC_InetAtoN, SOC_InetNtoA, SOC_InetNtoP
