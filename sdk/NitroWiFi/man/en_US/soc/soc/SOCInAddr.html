SOCInAddr

C Specification

#include <nitroWiFi/soc.h>
#define SOC_INADDR_ANY       ((u32) 0x00000000)
#define SOC_INADDR_BROADCAST ((u32) 0xffffffff)
#define SOC_INADDR_LOOPBACK  ((u32) 0x7f000001)  // 127.0.0.1
typedef struct SOCInAddr
{
u32 addr;
} SOCInAddr;

Description

SOCInAddr is used to store the IPv4 address.
