SOC_U32to4U8

C Specification

#include <nitroWiFi/soc.h>
void SOC_U32to4U8( u32 adr_u32, u8* adr_4u8 );

Arguments

adr_u32 The u32 value to assign.
adr_4u8 The output destination array.

Return Values

None.

Description

Converts u32 values to a u8 x 4 array.
