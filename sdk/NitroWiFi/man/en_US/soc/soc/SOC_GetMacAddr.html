SOC_GetMacAddr

C Specification

#include <nitroWiFi/soc.h>
void SOC_GetMacAddr ( SOCInterface* interface, u8* macAddress );

Arguments

interface The pointer to the SOCInterface that obtains the MAC address. This argument is included for compatibility issues, but it is ignored in actual use of the function.
macAddress The pointer to the buffer used for receiving the MAC address.

Return Values

None.

Description

Gets the MAC (media access control) address for the specified interface.
