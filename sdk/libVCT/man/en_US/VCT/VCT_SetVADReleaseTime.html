VCT_SetVADReleaseTime

Syntax

#include <vct.h>
void VCT_SetVADReleaseTime( int count );

Argument

count Release time (number of packets).

Return Value

None.

Description

VCT_SetVADReleaseTime sets the duration of time where packets are not sent if VAD determines that the audio data is muted.

A value of 500ms or longer is recommended to prevent false mute detection during short periods of silence, such as a short breath in between sentences.

This function's setting defines the amount of time to wait before stopping transmission upon detecting silence. The time is counted using the number of packets in count as units. For example, setting to 10 packets defines the release time as 680ms (68 x 10). The default setting is 15 packets (68 x 15 = 1020ms).
