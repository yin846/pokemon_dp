VCT_ResetVAD

Syntax

#include <vct.h>
void VCT_ResetVAD( void );

Argument

None.

Return Value

None.

Description

VCT_ResetVAD resets the internal condition of VAD. This function should be called after adjusting the VAD parameters (such as mic gain).
