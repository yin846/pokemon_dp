VCTMode

Definition

#include <vct.h>

typedef enum {
    VCT_MODE_NULL,
    VCT_MODE_PHONE,
    VCT_MODE_TRANSCEIVER,
    VCT_MODE_CONFERENCE
} VCTMode;

Definition

VCT_MODE_NULL The mode has not been set.
VCT_MODE_PHONE Telephone Mode.
VCT_MODE_TRANSCEIVER Transceiver Mode.
VCT_MODE_CONFERENCE Conference Mode.

Description

These are enumeration type functions to show VoiceChat Library's present mode.

VCT_Init, VCTConfig
