VCT_DeleteSession

Syntax

#include <vct.h>
BOOL VCT_DeleteSession( VCTSession *session );

Argument

session VCTSession structure to be deleted.

Return Value

TRUE Success.
FALSE Failure.

Description

VCT_DeleteSession deletes a session.
