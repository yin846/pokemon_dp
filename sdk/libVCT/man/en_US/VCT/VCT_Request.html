VCT_Request

Syntax

#include <vct.h>
int VCT_Request( VCTSession *session, VCTRequestCode request );

Argument

session A session to issue a request.
request A request code.

Return Value

0 if successful; otherwise, the error code returns as a negative number.

Description

VCT_Request issues a request of SSP.

In Transceiver Mode, the VCT_Contact function and the VCT_Release function should be used instead of VCT_Request.

See Also

VCT_CreateSession, VCT_Response
