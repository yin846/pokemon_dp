VCTEventCallback

Definition

#include <vct.h>

typedef void (*VCTEventCallback)(u8 aid,
                                 VCTEvent result,
                                 VCTSession *session,
                                 void *param);

Definition

aid AID of an event-generating remote machine.
result An event code.
session An event session.
param Parameters set by a user.

Description

VCTEventCallback is an event callback function in VoiceChat Library. The event callback is specified by the VCTConfig function.

This function is called from the VCT_HandleData and VCT_Main functions.

See Also

VCT_Init, VCTConfig
