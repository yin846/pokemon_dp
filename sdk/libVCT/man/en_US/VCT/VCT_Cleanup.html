VCT_Cleanup

Syntax

#include <vct.h>
void VCT_Cleanup( void );

Argument

None.

Return Value

None.

Description

VCT_Cleanup performs a clean-up operation of VoiceChat Library. This function is called upon exiting a program or changing VoiceChat Library operation modes.
