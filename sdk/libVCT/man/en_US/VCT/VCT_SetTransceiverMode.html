VCT_SetTransceiverMode

Syntax

#include <vct.h>
void VCT_SetTransceiverMode( BOOL true_or_false );

Argument

true_or_false TRUE if set to server, FALSE if set to client.

Return Value

None.

Description

VCT_SetTransceiverMode determines whether a machine acts as a server or a client in Transceiver Mode. This function is only available in Transceiver Mode.

See Also

VCT_SetTransceiverServer, VCT_SetTransceiverClients
