VCT_GetVADInfo

Syntax

#include <vct.h>
void VCT_GetVADInfo( VCTVADInfo *outInfo );

Argument

outInfo A structure pointer that stores the internal parameters of VAD.

Return Value

None.

Description

VCT_GetVADInfo retrieves the internal parameters of VAD.

See Also

VCT_VADInfo, VCT_SetVADReleaseTime
