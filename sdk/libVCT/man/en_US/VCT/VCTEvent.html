VCTEvent

Definition

#include <vct.h>

typedef enum {
    VCT_EVENT_NONE,
    
    VCT_EVENT_INCOMING,
    VCT_EVENT_REJECT,
    VCT_EVENT_BUSY,
    VCT_EVENT_CANCEL,
    VCT_EVENT_NOTIFY_FREE,
    VCT_EVENT_NOTIFY_BUSY,
    VCT_EVENT_CONNECTED,
    VCT_EVENT_RESPONDBYE,
    VCT_EVENT_DISCONNECTED,
    VCT_EVENT_CONTACT,
    VCT_EVENT_TIMEOUT,
    VCT_EVENT_ABORT
} VCTEvent;

Definition

VCT_EVENT_NONE There is no event occurrence.
VCT_EVENT_INCOMING A request to talk was received.
VCT_EVENT_REJECT A request to talk was rejected.
VCT_EVENT_BUSY The receiving party is busy.
VCT_EVENT_CANCEL A request to talk was canceled.
VCT_EVENT_NOTIFY_FREE Another player stopped a conversation in Transceiver Mode.
VCT_EVENT_NOTIFY_BUSY Another player started a conversation in Transceiver Mode.
VCT_EVENT_CONNECTED The request for conversation was accepted.
VCT_EVENT_RESPONDBYE The request for terminating the conversation was received.
VCT_EVENT_DISCONNECTED The process to terminate the conversation.
VCT_EVENT_CONTACT A request to talk was received in Transceiver Mode.
VCT_EVENT_TIMEOUT A session time-out occurred.
VCT_EVENT_ABORT A session was aborted by events such as invalid processing.

Description

These are enumeration functions of VoiceChat Library event code.
