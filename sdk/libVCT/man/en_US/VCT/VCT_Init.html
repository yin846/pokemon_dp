VCT_Init

Syntax

#include <vct.h>
BOOL VCT_Init( VCTConfig *config );

Argument

config VCTConfig structure

Return Value

TRUE Initialization succeeds.
FALSE Initialization fails

Description

VCT_Init initializes VoiceChat Library.VCTConfigStructure must be properly configured for normal operation.

Any change to the Library's operation mode requires calling VCT_Cleanup first, then calling VCT_Init again.

See Also

VCT_Cleanup, VCTConfig
