VCT_Response

Syntax

#include <vct.h>
int VCT_Response(VCTSession *session, VCTResponseCode response);
Argument
session A session to issue a response.
response Response code.

Return Value

0 if successful; otherwise, the error code returns as a negative number.

Description

VCT_Response issues a response of SSP.

See Also

VCT_CreateSession, VCTRequest
