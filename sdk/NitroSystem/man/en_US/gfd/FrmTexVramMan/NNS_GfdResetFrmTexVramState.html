NNS_GfdResetFrmTexVramState

C Specification

#include <nnsys/gfd.h>
void NNS_GfdResetFrmTexVramState(void);

Arguments

None.

Return Values

None.

Description

Resets the texture-use memory allocation state to the initial state. This operation frees all texture-use memory allocated from the Frame Texture VRAM Manager.
