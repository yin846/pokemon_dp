NNS_GfdGetVramTransferTaskTotalSize

C Specification

#include <nnsys/gfd.h>
u32 NNS_GfdGetVramTransferTaskTotalSize(void);

Arguments

None.

Return Values

The total data-transfer size for the VRAM transfer tasks.

Description

Gets the total data-transfer size of the VRAM transfer tasks registered in the VRAM Transfer Manager.
