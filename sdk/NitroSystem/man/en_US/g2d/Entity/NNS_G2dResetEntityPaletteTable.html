NNS_G2dResetEntityPaletteTable

C Specification

#include <nnsys/g2d/g2d_Entity.h>
void NNS_G2dResetEntityPaletteTable( NNSG2dEntity* pEntity );

Arguments

pEntity [OUT] Entity

Return Values

None.

Description
