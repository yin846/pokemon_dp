NNS_G2dSetEntityCurrentAnimation

C Specification

#include <nnsys/g2d/g2d_Entity.h>
void NNS_G2dSetEntityCurrentAnimation( NNSG2dEntity* pEntity, u16 idx );

Arguments

pEntity [OUT] Entity
idx [IN] Playback animation number

Return Values

None.

Description

Sets the entity playback animation. No action will occur if an invalid index is input. The name of this function was changed to NNS_G2dSetCurrentAnimation. This function will remain to preserve compatability.

See Also

NNS_G2dSetCurrentAnimation, NNS_G2dInitEntity
