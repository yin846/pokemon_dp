NNS_G2dUnPack*

C Specification

#include <nnsys/g2d/fmt/g2d_NAN_load.h>
#include <nnsys/g2d/fmt/g2d_NCE_load.h>
#include <nnsys/g2d/fmt/g2d_NEN_load.h>
#include <nnsys/g2d/fmt/g2d_NMC_load.h>
void NNS_G2dUnpackNAN( NNSG2dAnimBankData* pData );
void NNS_G2dUnpackNCE( NNSG2dCellDataBank* pCellData );
void NNS_G2dUnpackNEN( NNSG2dEntityDataBank* pDataBank );
void NNS_G2dUnpackNMC( NNSG2dMultiCellDataBank* pMCellData );

Arguments

pData [IN, OUT] Animation bank data
pCellData [IN, OUT] Cell bank data
pDataBank [IN, OUT] Entity bank data
pMCellData [IN, OUT] Multicell sequence data

Return Values

None.

Description

Unpacks binary data loaded from a file to a usable state. This function must be executed before any binary data is used. Execute this function only once for the data.
Internally, extraction occurs to the memory offset pointer value.
