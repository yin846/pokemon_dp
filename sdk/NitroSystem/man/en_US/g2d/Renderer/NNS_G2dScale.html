NNS_G2dScale

C Specification

#include <nnsys/g2d/g2d_Renderer.h>
void NNS_G2dScale( fx32 x,  fx32 y,  fx32 z );

Arguments

x [IN] Scale along x
y [IN] Scale along y
z [IN] Scale along z

Return Values

None.

Description

Multiplies the scale matrix by the renderer's current internal matrix stack.

See Also

NNS_G2dTranslate, NNS_G2dRotZ
