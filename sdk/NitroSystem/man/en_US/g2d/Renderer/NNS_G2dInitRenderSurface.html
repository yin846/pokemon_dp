NNS_G2dInitRenderSurface

C Specification

#include <nnsys/g2d/g2d_Renderer.h>
void NNS_G2dInitRenderSurface( NNSG2dRenderSurface* pSurface );

Arguments

pSurface [IN] Render surface entity

Return Values

None.

Description

Initializes the render surface. This function must be called before using the render surface.
