NNS_G2dTextCanvasSetVSpace

C Specification

#include <nnsys/g2d/g2d_TextCanvas.h>

void NNS_G2dTextCanvasSetVSpace(
	NNSG2dTextCanvas* pTxn,
	int vSpace
);

Arguments

pTxn [IN] Pointer to TextCanvas
vSpace [IN] New setting for line spacing (in pixels)

Return Values

None.

Description

Changes the line spacing used by TextCanvas.
