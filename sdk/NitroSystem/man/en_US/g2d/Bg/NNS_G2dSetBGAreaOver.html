NNS_G2dSetBGAreaOver

C Specification

#include <nnsys/g2d/g2d_Screen.h>
void NNS_G2dSetBGAreaOver( GXBGAreaOver areaOver );

Arguments

areaOver The area-over process specifications used by NNS_G2dBGSetup.

Return Values

None.

Description

Sets the out-of-area process specifications used by NNS_G2dBGSetup() for configuring the BG control settings for Affine (extended) BG. By default, NNS_G2dBGSetup() specifies GX_BG_AREAOVER_XLU as the out-of-area process.

See Also

NNS_G2dBGSetup, NNS_G2dGetBGAreaOver
