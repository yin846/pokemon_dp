NNS_G2dGetBGAreaOver

C Specification

#include <nnsys/g2d/g2d_Screen.h>
GXBGAreaOver NNS_G2dGetBGAreaOver( void );

Arguments

None.

Return Values

The out-of-area process specifications to be used by NNS_G2dBGSetup().

Description

This function obtains the out-of-area process specifications used by NNS_G2dBGSetup() when configuring the BG control settings for Affine (extended) BG.

See Also

NNS_G2dBGSetup, NNS_G2dSetBGAreaOver
