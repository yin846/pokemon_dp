NNS_G2dEntryNewOamWithAffineIdx

C Specification

#include <nnsys/g2d/g2d_Oam.h>
BOOL NNS_G2dEntryNewOamWithAffineIdx
                      ( 
        NNSG2dOamManagerInstance*       pMan, 
        const GXOamAttr*                pOam, 
        u16                         affineIdx );

Arguments

pMan [OUT] OAM manager entity
pOam [IN] GXOamAttr address to be registered
affineIdx [IN] Index of the affine parameters used by the GXOamAttr to be registered

Return Values

If there is not enough free space, nothing is registered and FALSE is returned.

Description

Registers OBJ in the OAM manager. After the registration occurs, the affine index that the OBJ references is rewritten. The name of this function has changed to NNS_G2dEntryOamManagerOamWithAffineIdx. To maintain compatibility, the previous function was kept under a different name.

See Also

NNS_G2dEntryOamManagerOamWithAffineIdx, NNS_G2dEntryOamManagerOam
