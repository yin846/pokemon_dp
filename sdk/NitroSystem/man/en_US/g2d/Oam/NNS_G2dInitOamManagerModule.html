NNS_G2dInitOamManagerModule

C Specification

#include <nnsys/g2d/g2d_Oam.h>
void NNS_G2dInitOamManagerModule()

Arguments

None.

Return Values

None.

Description

Initializes the OAM Manager module. Call this function before executing any OAM Manager-related functions.

The function acts internally to initialize OAM-use region tables. NNS_G2dInitializeOamManager() has undergone a name change. This function will remain to preserve compatability.

See Also

NNS_G2dInitializeOamManager, NNS_G2dGetNewManagerInstance
