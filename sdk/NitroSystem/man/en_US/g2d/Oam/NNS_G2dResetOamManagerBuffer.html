NNS_G2dResetOamManagerBuffer

C Specification

#include <nnsys/g2d/g2d_Oam.h>
void NNS_G2dResetOamManagerBuffer( NNSG2dOamManagerInstance* pMan );

Arguments

pMan [OUT] OAM manager entity

Return Values

None.

Description

Resets the internal OAM buffer for the range that the OAM manager is managing. (It is forcibly overwritten with the default value that sets the OBJ outside the screen.) The name has changed from NNS_G2dResetBuffer. This function will remain to preserve compatibility.

See Also

NNS_G2dResetBuffer, NNS_G2dGetOamManagerOamCapacity, NNS_G2dGetOamManagerAffineCapacity
