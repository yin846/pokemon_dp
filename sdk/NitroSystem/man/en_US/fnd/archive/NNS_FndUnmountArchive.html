NNS_FndUnmountArchive

C Specification

#include <nnsys/fnd.h>
BOOL NNS_FndUnmountArchive(NNSFndArchive* archive);

Arguments

archive The pointer to the NNSFndArchive structure.

Return Values

TRUE if the archive has been unmounted successfully.

Description

This function unmounts the specified archive currently mounted to the file system.
