NNS_McsPutString

C Specification

#include <nnsys/mcs.h>
BOOL NNS_McsPutString(const char* string);

Arguments

string Character string to output.

Return Values

Returns TRUE when the function was successful. Returns FALSE if it failed.

Description

Outputs a character string to the MCS server.
