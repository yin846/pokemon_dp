NNS_SndEndSleep

C Specification

#include <nnsys/snd.h>
void NNS_SndEndSleep( void );

Arguments

None.

Return Values

None.

Description

It is not necessary to call this function to end sleep. When this function is called, it no longer does anything.

See Also

None.
