NNS_SndPlayerGetSeqArcNo

C Specification

#include <nnsys/snd.h>
int NNS_SndPlayerGetSeqArcNo( NNSSndHandle* handle );

Arguments

handle The sound handle pointer.

Return Values

Returns the sequence archive number.

Description

Gets the sequence archive number.

This function gets the sequence archive number of the sequence that is bound to the sound handle handle. If the sound handle is invalid, a value of -1 will be returned.

The information that can be obtained is based on the value set by the NNS_SndPlayerSetSeqArcNo function. When nothing is set, a -1 will be returned.
