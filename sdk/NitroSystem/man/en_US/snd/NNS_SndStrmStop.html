NNS_SndStrmStop

C Specification

#include <nnsys/snd.h>
void NNS_SndStrmStop( NNSSndStrm* stream );

Arguments

stream Pointer to a stream structure.

Return Values

None.

Description

Stops steam playback.

Note

To resume the stream playback, the NNS_SndStrmSetup function needs to be called again.

See Also

NNS_SndStrmSetup
