NNS_SndMain

C Specification

#include <nnsys/snd.h>
void NNS_SndMain( void );

Arguments

None.

Return Values

None.

Description

This function performs the frame processing of sound. Call once for every game frame. The timing of the call is irrelevant.

Internally, the parameters are updated, and communication occurs with ARM7.
