NNS_SndArcGetSeqArcInfo

C Specification

#include <nnsys/snd.h>
const NNSSndArcSeqArcInfo* NNS_SndArcGetSeqArcInfo(
int seqArcNo );

Arguments

seqArcNo Sequence archive number

Return Values

Returns a pointer to the sequence archive structure NNSSndArcSeqArcInfo. Returns a NULL if an invalid sequence archive number was assigned.

Description

Gets the sequence archive information structure.
