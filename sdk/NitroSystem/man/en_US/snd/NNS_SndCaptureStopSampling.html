NNS_SndCaptureStopSampling

C Specification

#include <nnsys/snd.h>
void NNS_SndCaptureStopSampling( void );

Arguments

None.

Return Values

None.

Description

Stops the sampling process. If sampling is not happening at the time, the function does nothing.
