NNS_SndCaptureIsActive

C Specification

#include <nnsys/snd.h>
BOOL NNS_SndCaptureIsActive( void );

Arguments

None.

Return Values

Returns whether or not the capture is executing.

Description

Obtains information as to whether the capture is currently executing or not.
