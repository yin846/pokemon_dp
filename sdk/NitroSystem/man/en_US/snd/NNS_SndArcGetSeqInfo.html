NNS_SndArcGetSeqInfo

C Specification

#include <nnsys/snd.h>
const NNSSndArcSeqInfo* NNS_SndArcGetSeqInfo( int seqNo );

Arguments

seqNo Sequence number

Return Values

Returns a pointer to the sequence information structure NNSSndArcSeqInfo. Returns a NULL if an invalid sequence number was assigned.

Description

Gets the sequence information structure.
