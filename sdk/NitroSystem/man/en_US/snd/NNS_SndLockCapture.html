NNS_SndLockCapture

C Specification

#include <nnsys/snd.h>
BOOL NNS_SndLockCapture( u32 capBitFlag );

Arguments

capBitFlag The bit flag of the capture to lock.

Return Values

Returns TRUE if the lock was successful. Returns FALSE if it fails.

Description

Locks the capture. The locked capture will no longer be used inside the library.

Designate the capture to lock with capBitFlag. In order from the lower bits, show capture 0 and capture 1, then simultaneously lock all captures with active bits.

The function will fail if you try to lock a capture that is already in use. Also, if you are trying to lock both captures at the same time and one of the locks fails, both locks will fail.

Call the NNS_SndUnlockCapture function in order to unlock a capture.
