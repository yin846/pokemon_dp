NNS_SndCaptureStopEffect

C Specification

#include <nnsys/snd.h>
void NNS_SndCaptureStopEffect( void );

Arguments

None.

Return Values

None.

Description

Stops the effect. If the output effect process is not running, this function does nothing.

Note

If this function is called while sound is being played, the sound is interrupted.

See Also

None.
