NNS_G3D_SBC_PRJMAP

Command Format

Offset (bytes) Description
0 NNS_G3D_SBC_PRJMAP | NNS_G3D_SBCFLG_000
1 The material ID.
2 Flag for expansion (Currently always set to 0).

Description

When the texture coordinate conversion mode designated by the NITRO intermediate file .imd format <material>::tex_gen_mode is pos, the NNS_G3D_SBC_PRJMAP command is configured immediately after the NNS_G3D_SBC_MAT command. This command calculates and configures the texture matrix for the projection map. Depending on the callback function, each of the texture matrix calculation operations can be replaced.

Timing for Calling a Callback Function Usage
NNS_G3D_SBC_CALLBACK_TIMING_A Allows for the customization of the calculation that maps the vertex coordinates to the texture coordinates.
NNS_G3D_SBC_CALLBACK_TIMING_B Allows for the customization of the calculation that performs the SRT conversion and projection conversion of the vertex coordinates in a world coordinate system.
NNS_G3D_SBC_CALLBACK_TIMING_C Allows for the customization of the calculation that is used to convert the input vertex coordinates to a world coordinate system.
