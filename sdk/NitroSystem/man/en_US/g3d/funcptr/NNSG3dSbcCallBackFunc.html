NNSG3dSbcCallBackFunc

C Specification

#include <nnsys/g3d/kernel.h>
typedef void (*NNSG3dSbcCallBackFunc)(struct NNSG3dRS_*);

Description

This is a type of callback function that can be called when the NNS_G3dDraw function is executed.
