NNSG3dFuncSbcShpTable

C Specification

#include <nnsys/g3d/sbc.h>

typedef void (*NNSG3dFuncSbc_ShpInternal) (NNSG3dRS*, u32, const NNSG3dResShpData*, u32);

NNSG3dFuncSbc_ShpInternal  NNS_G3dFuncSbcShpTable[NNS_G3D_SIZE_SHP_VTBL_NUM];

Description

This function table is used to allocate the processes of the SBC NNS_G3D_SBC_SHP commands to each material data format. Since there is currently 1 shape data format, this is only a specification for future expansion.
