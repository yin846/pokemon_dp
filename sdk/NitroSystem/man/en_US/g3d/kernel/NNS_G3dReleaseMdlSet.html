NNS_G3dReleaseMdlSet

C Specification

#include <nnsys/g3d/kernel.h>
void
NNS_G3dReleaseMdlSet(
NNSG3dResMdlSet* pMdlSet
);

Arguments

pMdlSet The pointer to the model set

Return Values

None.

Description

Releases the association between the texture and palette from every model in the set pMdlSet.
