NNS_G3dRenderObjSetBlendFuncVis

C Specification

#include <nnsys/g3d/anm.h>

void
NNS_G3dRenderObjSetBlendFuncVis(
NNSG3dRenderObj* pRenderObj,
NNSG3dFuncAnmBlendVis func
);

Arguments

pRenderObj Pointer to the NNSG3dRenderObj structure
func The pointer to the blend function of the visibility calculation result

Return Values

None.

Description

Configures the visibility calculation results' blend function in the rendering object. The blend function configured here is called from G3D when rendering with the NNS_G3dDraw function. Because the default blend function is set in the rendering object when the rendering object is initialized, there is no need to call this function when using the default blend function.
