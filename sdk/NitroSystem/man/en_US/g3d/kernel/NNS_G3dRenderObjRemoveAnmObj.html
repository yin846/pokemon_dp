NNS_G3dRenderObjRemoveAnmObj

C Specification

#include <nnsys/g3d/kernel.h>
void
NNS_G3dRenderObjRemoveAnmObj(
NNSG3dRenderObj* pRenderObj,
NNSG3dAnmObj* pAnmObj
);

Arguments

pRenderObj Pointer to the NNSG3dRenderObj structure
pAnmObj Pointer to the NNSG3dAnmObj structure

Return Values

None.

Description

Deletes the animation object designated by pAnmObj from the rendering object designated by pRenderObj.
