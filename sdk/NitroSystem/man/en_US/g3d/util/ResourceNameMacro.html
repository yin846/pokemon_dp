Resource Name Macro

Description

NNS_G3D_DEFRESNAME(var, str) Defines the resource name var initialized by the character string str.
NNS_G3D_GET_JNTID(pMdl, pJntID, literal) Searches for the node used as the resource name for the character string literal from the model pMdl. The index of the node that is found is configured in *pJntID.
NNS_G3D_GET_MATID(pMdl, pMatID, literal) Searches for the material used as the resource name for the character string literal from the model pMdl. The index of the material that is found is configured in *pMatID.
NNS_G3D_GET_SHPID(pMdl, pShpID, literal) Searches for the shape used as the resource name for the character string literal from the model pMdl. The index of the shape that is found is configured in *pShpID.
NNS_G3D_GET_MDL(pMdlSet, pMdl, literal) Searches for the model used as the resource name for the character string literal from the model set pMdlSet. The pointer to the model that is found is configured in pMdl.
NNS_G3D_GET_ANM(pRes, pResAnm, literal) Searches for the animation resource from the loaded animation resource file by using the string literal as a resource name, and sets the pointer to the resource that is found to pResAnm.
