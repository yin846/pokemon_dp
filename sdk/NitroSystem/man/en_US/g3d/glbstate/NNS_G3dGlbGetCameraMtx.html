NNS_G3dGlbGetCameraMtx

C Specification

#include <nnsys/g3d/glbstate.h>
const MtxFx43* NNS_G3dGlbGetCameraMtx( void );

Arguments

None.

Return Values

The pointer to the camera matrix.

Description

Gets the pointer to the camera matrix, which is set in the global state stored in the G3D library.
