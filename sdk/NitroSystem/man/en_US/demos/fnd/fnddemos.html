Foundation Library Sample Demo

Location

$NitroSystem/build/demos/fnd

Compilation

Move to the $NitroSystem/build/demos/fnd and enter

% make

 Executable (.nef, .srl) files will be generated in

$NitroSystem/build/demos/fnd/sample_name/bin/platform/target

 For example:

$NitroSystem/build/demos/fnd/archive/bin/ARM9-TS/Release/

 

Execution

Move to the $NitroSystem/build/demos/fnd/sample_name/bin/platform/target/ directory and do one of the following:

  • Execute *.srl with ensata.
  • Load and execute a *.nlf file with IS-NITRO-DEBUGGER.
  • Execute a *.srl file with IS-NITRO-DEBUGGER.

Description


archive Reads in the BG data from the archive and displays it on the screen.
archive2 Reads in the BG data from the compressed archive and displays it on the screen.
heap This sample shows simple uses for expanded, frame, and unit heap. Nothing will be displayed on the screen, but the heap status will be displayed in the debug output.

Operating Environment

Normal operation for all of the samples using the IS-NITRO-EMULATOR and ensata have been confirmed.
