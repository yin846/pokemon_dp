DWC_NASLoginAsync

C Specification

#include <dwc.h>

BOOL DWC_NASLoginAsync(void);

Description

This function performs authentication with the Nintendo Authentication Server and connects to the Nintendo Wi-Fi Connection. After calling the function, you will need to call the DWC_NASLoginProcess function to continue.

Before calling this function, the connection to the Internet must be complete.
Also, there is no need to call this function when using the FriendsMatch Library, since the same sort of processing is performed by the DWC_LoginAsync function.

Arguments

None.

Return Values

TRUE The authentication process has started.
FALSE The authentication process could not start.

See Also

DWC_NASLoginProcess, FriendsMatch Library, DWC_LoginAsync
