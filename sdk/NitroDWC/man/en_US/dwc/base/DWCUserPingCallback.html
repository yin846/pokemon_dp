DWCUserPingCallback

C Specification

#include <dwc.h>

typedef void (*DWCUserPingCallback)( int latency, u8 aid );

Description

This callback function notifies the user of the measurement results from the DWC_Ping function.

This callback function can be set using DWC_SetUserPingCallback.

Arguments

latency Latency associated with ping (in milliseconds).
aid AID of remote host for which ping was measured.

Return Values

None.

See Also

DWC_Ping, DWC_SetUserPingCallback
