General-Purpose Ranking Constants

Definition

#include <dwc.h>

#define DWC_RNK_DATA_MAX        768

#define DWC_RNK_GET_MAX         10

#define DWC_RNK_CATEGORY_MAX    100

#define DWC_RNK_ORDER_ASC       0
#define DWC_RNK_ORDER_DES       1

Specific to DWCRnkPutScoreAsync
and DWCRnkGetScoreAsync

DWC_RNK_DATA_MAX The maximum size available for user-defined data.
DWC_RNK_GET_MAX The maximum value in the retrieved ranking list.
DWC_RNK_CATEGORY_MAX The maximum value that can be specified as a category.
DWC_RNK_ORDER_ASC The constant for showing the retrieved ranking list in ascending order.
DWC_RNK_ORDER_DES The constant for showing the retrieved ranking list in descending order.
