DWC_LoadOwnDataAsync

C Specification


#include <dwc.h>

BOOL   DWC_LoadOwnDataAsync( char* keys, void* param );

Description

This function is retained to maintain compatibility with older versions. This function can load only one's own data saved with the DWC_SavePrivateDataAsync function.

Use the DWC_LoadOwnPrivateDataAsync function, as it performs the same function.

Arguments

keys Pointer to the key string for the data to load.
param Callback parameter

Return Values

TRUE Starting load.
FALSE Cannot load because either the DS is offline or an error is being generated.

See Also

DWC_SavePublicDataAsync, DWC_SavePrivateDataAsync, DWC_LoadOwnPublicDataAsync, DWC_LoadOwnPrivateDataAsync
