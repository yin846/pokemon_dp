PRC_IsEmpty

C Specification

#include <nitro/prc.h>

BOOL PRC_IsEmpty( PRCStrokes* strokes );

Arguments

strokes Pointer to the input stroke structure.

Return Values

Returns TRUE if strokes is empty.

Description

Checks to see if strokes is empty.
