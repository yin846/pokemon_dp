PRC_IsFull

C Specification

#include <nitro/prc.h>

BOOL PRC_IsFull( PRCStrokes*  strokes );

Arguments

strokes Pointer to the input stroke structure.

Return Values

Returns TRUE if strokes is full.

Description

Checks whether strokes is full.
