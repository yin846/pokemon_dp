PRCPoint

Definitions

#include <nitro/prc.h>

typedef struct PRCPoint
{
s16                 x;
s16                 y;
} PRCPoint;

Elements

x X coordinate value.
y Y coordinate value.

Description

Holds coordinate values that are input from the touch panel.
