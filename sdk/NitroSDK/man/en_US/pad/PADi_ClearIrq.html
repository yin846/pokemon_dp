PADi_ClearIrq

C Specification

#include <nitro/pad/pad.h>

void PADi_ClearIrq( void );

Arguments

None.

Return Values

None.

Description

This function prohibits interrupts corresponding to key input. This function zero-clears the IO register REG_KEYCNT (Key Control).

See Also

PAD_Read, PADi_SetIrq
