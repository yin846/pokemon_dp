PAD_DetectFold

C Specification

#include <nitro/pad.h>

BOOL PAD_DetectFold( void );

Arguments

None.

Return Values

Returns TRUE when the cover is folded. Returns FALSE when open.

Description

Detects whether or not the cover is folded (closed)
