WM_PowerOff

C Specification

#include <nitro/wm.h>

WMErrCode WM_PowerOff( WMCallbackFunc callback );

Arguments

callback Designates the callback function to call when the asynchronous process completes. This callback is called from within the interrupt handler.

Return Values

Returns the results of the WMErrCode enumerated type process.

Description

Shuts down the wireless hardware and stops the power to the WM If the return value is WM_ERRCODE_OPERATING, a callback is generated after the shutdown process completes. This function can only be called when the WM is in the IDLE state. As a result of this function, the state changes to STOP. After calling this function, use WM_Disable to return the WM hardware to the disabled state.

See Also

WM_Init, WM_Enable, WM_PowerOn, WM_Disable, WM_Finish, WM_Initialize, WM_End
