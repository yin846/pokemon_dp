WM_GetAID

C Specification

#include <nitro/wm.h>

u16 WM_GetAID(void);

Arguments

None.

Return Values

Returns the current AID.

Description

Gets the current AID. Returns a valid value only when in the PARENT, MP_PARENT, CHILD, or MP_CHILD state.
