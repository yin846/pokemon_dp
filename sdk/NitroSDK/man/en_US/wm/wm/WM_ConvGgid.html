WM_ConvGgid*

C Specification

#include <nitro/wm.h>

void WM_ConvGgid32to16( u32* src , u16* dst );
void WM_ConvGgid16to32( u16* src , u32* dst );

Arguments

src Designates the pointer to the conversion source variable of GGID.
dst Designates the pointer to the buffer that stores the post-conversion data.

Return Values

None.

Description

The 32-bit Game Group ID (GGID) can be expressed as u32 type or as an array of the u16 type. This function converts the GGID from one format to another.
WM_ConvGgid32to16 converts the u32 type GGID to a u16 type array. WM_ConvGgid16to32 converts the u16 type array GGID to a u32 type.
