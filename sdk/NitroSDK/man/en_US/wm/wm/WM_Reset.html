WM_Reset

C Specification

#include <nitro/wm.h>

WMErrCode WM_Reset( WMCallbackFunc callback );

Arguments

callback Designates the callback function to call when the asynchronous process completes. This callback is called from within the interrupt handler.

Return Values

Returns the results of the WMErrCode enumerated type process.

Description

Resets the WM This function changes the WM from any state other than READY and STOP to IDLE.
Connection information that has been stored until this time will be deleted. If the return value for the function is WM_ERRCODE_OPERATING, a callback is triggered after the reset process is complete.

See Also

WM_Initialize, WM_SetIndCallback, WM_End
