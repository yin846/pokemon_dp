WM_EndKeySharing

C Specification

#include <nitro/wm.h>

WMErrCode WM_EndKeySharing( WMKeySetBuf* buf );

Arguments

buf Pointer to the buffer that stores key information. The entity is the pointer to the WMDataSharingInfo structure.

Return Values

Returns the results of the WMErrCode enumerated type process.

Description

Disables the key sharing feature. This function is executable only when the key sharing feature is enabled by WM_StartKeySharing. This function was changed from asynchronous to synchronous. WM_StartKeySharing, WM_EndKeySharing, and WM_GetKeySet remain for backward compatibility, but they will be removed in the future. Use the Data Sharing functions instead.

See Also

WM_StartKeySharing, WM_GetKeySet
