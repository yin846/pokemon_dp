WM_EndDCF

C Specification

#include <nitro/wm.h>

WMErrCode WM_EndDCF( WMCallbackFunc callback );

Arguments

callback Designates the callback function to call when the asynchronous process completes. This callback is called from within the interrupt handler.

Return Values

Returns the results of the WMErrCode enumerated type process.

Description

Ends infrastructure mode communications. This function is executable only when the WM is in the DCF CHILD state. If the return value for this function is WM_ERRCODE_OPERATING, a callback is generated after the disconnection process with the infrastructure mode is complete. The WM returns to the CHILD state.

See Also

WM_StartDCF, WM_SetDCFData
