MB_GetTgid

C Specification

#include <nitro/mb.h>

u16 MB_GetTgid(void);

Arguments

None.

Return Values

Returns the TGID value specified by the MB_Init function.

Description

Gets value of TGID specified by the MB_Init function. This value cannot be changed after it is set, but the acquisition function has been provided for your convenience.


See Also

MB_Init, MB_End
