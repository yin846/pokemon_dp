MB_TGID_AUTO

Definitions

#include <nitro/mb.h>

/* Definition value that specifies the automatic setting of TGIDs in MB_Init().
#define  MB_TGID_AUTO   0x10000

Description

This constant indicates that TGIDs should be automatically generated in MB_Init. This constant is defined as a value that does not correspond to a 16-bit value between 0x0000 and 0xFFFF that indicates a proper TGID value.
