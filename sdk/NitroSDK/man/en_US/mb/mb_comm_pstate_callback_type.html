MBCommPStateCallback

Definitions

#include <nitro/mb.h>

typedef void (*MBCommPStateCallback)(u16 child_aid, u32 status, void* arg);

Description

Defines the format of the callback function, which is called every time the parent state changes. The arguments are described below:
Arguments Description
child_aid When a callback is a notification for one specific child, shows the aid of that child. Specify this value and call the MB_CommGetChildUser function to get the details regarding a specific child.
status Indicates the state shown in MBCommPState.
arg The variable corresponding to the state to be notified is passed. For details on each status, see the description of MBCommPState.


See Also

MBCommPState, MB_CommGetChildUser, MB_CommSetParentStateCallback, MB_CommGetParentState
