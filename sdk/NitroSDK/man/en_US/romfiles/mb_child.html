mb_child

Overview

mb_child is the program that extracted only the DS download play menu included in the release version startup program (IPL). This program downloads programs without checking the authentication codes, so use it for testign during development.

Operation Method

When this program is started, the phrase "Searching for downloadable software" appears.  After that, every time a DS Download Play parent is found, the icon and information such as game names are added one by one. Conversely, if a parent device can no longer be detected after a fixed amount of time, its information is removed.
Select a game with the Up and Down keys on the +Control Pad, and begin the download with the A Button.
After that, the phrase "Downloading..." will be displayed in the interval between the completion of the download and the receipt of a restart request from the parent device.  Pressing the B Button at this time cancels the download and returns to the initial selection screen.
There will be a restart in accordance with the request from the parent device, and then the downloaded game will start.

Location
