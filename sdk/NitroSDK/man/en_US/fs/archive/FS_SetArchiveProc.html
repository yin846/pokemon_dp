FS_SetArchiveProc

C Specification

#include <nitro/fs.h>

void FS_SetArchiveProc(struct FSArchive *p_arc, FS_ARCHIVE_PROC_FUNC proc, u32 flags);

Arguments

p_arc Address of the FSArchive structure
proc User procedure to register. The function set here will be called from inside the FS function that uses archive commands.
flags The bit set representing the types of commands sent to the user procedure. These are specified by a combination of FS_ARCHIVE_PROC_* flags.

Return Values

None.

Description

Specifies a user procedure in an archive. User procedures are called as callback functions when the archive processes commands or enters certain states. During the processing of the callback, the commands themselves can be replaced by independent processes or things can be left to the default implementation.
For more information about the operation of each command, see FSCommandType.

See Also

FSArchive, FSCommandType, FS_ARCHIVE_PROC_*, FS_LoadArchive, FS_UnloadArchive
