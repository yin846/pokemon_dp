FS_ResumeArchive

C Specification

#include <nitro/fs.h>

BOOL FS_ResumeArchive(FSArchive *p_arc);

Arguments

p_arc Address of the FSArchive structure

Return Values

FALSE if archive is in the suspended state before the change. TRUE if not.

Description

Returns the archive from the halted state to the normal state after any FS_SuspendArchive call. The archive is in the normal state at the time of initialization, and gets put in the stopped state by calling the FS_SuspendArchive function.
