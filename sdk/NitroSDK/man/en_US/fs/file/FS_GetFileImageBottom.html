FS_GetFileImageBottom

C Specification

#include <nitro/fs.h>

static inline u32 FS_GetFileImageBottom(const FSFile *p_file);

Arguments

p_file Address of the FSFile structure
It must retain the file information.

Return Values

Returns the end of the file image area's offset value in the archive.

Description

Returns the end of the file image area's offset value in the archive. The return value is undefined when this function is called by any files other than the one opened by the FS_OpenFile function.

See Also

FSFile, FS_GetFileImageTop, FS_OpenFile, FS_GetArchiveBase
