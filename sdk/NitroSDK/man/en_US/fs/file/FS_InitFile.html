FS_InitFile

C Specification

#include <nitro/fs.h>

void FS_InitFile( FSFile *p_file );

Arguments

p_file Address of the FSFile structure

Return Values

None.

Description

Initializes the FSFile structure. Call this once before using the FSFile structure.

See Also

FSFile, FS_OpenFile, FS_OpenFileFast
