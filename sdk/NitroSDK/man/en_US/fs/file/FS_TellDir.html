FS_TellDir

C Specification

#include <nitro/fs.h>

BOOL FS_TellDir( const FSFile *p_dir, FSDirPos *p_pos );

Arguments

p_dir Address of the FSFile structure
p_pos The address of the structure that stores the directory position.

Return Values

If the directory entry is legal, it stores the value of FSDirPos that indicates current position, and returns TRUE.
Otherwise it returns FALSE.

Description

This acquires the list position that is currently maintained in the specified directory list as FSDirPos. If you use this value when you call the FS_SeekDir function, you will be able to repeat the current list position once more.

See Also

FSFile, FSDirPos, FS_FindDir, FS_ReadDir, FS_SeekDir, FS_RewindDir
