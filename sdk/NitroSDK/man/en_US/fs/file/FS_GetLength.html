FS_GetLength

C Specification

#include <nitro/fs.h>

u32 FS_GetLength( const FSFile *p_file );

Arguments

p_file Address of the FSFile structure

Return Values

Returns the size of the file.

Description

This acquires the size of an open file.

See Also

FSFile, FS_OpenFile, FS_OpenFileFast, FS_CloseFile
