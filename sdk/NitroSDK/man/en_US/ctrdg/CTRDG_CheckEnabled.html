CTRDG_CheckEnabled

C Specification

#include <nitro/ctrdg.h>

void CTRDG_CheckEnabled(void);

Arguments

None.

Return Values

None.

Description

This function checks access permission status to determine whether access to the AGB Game Pak is enabled. The program will forcibly halt if an attempt is made to access the AGB Game Pak while access is disabled. Note that this access permission does not apply to direct AGB Game Pak operations that do not use the CTRDG library.
For functions affected by this access permission, see CTRDG_Enable.
The default value of the access permission status is FALSE, and may be changed using the CARD_Enable function.
