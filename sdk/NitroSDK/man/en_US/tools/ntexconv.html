ntexconv

Description

ntexconv is a tool for converting BMP or TGA format image files into the format that is used by NITRO libraries, etc.

ntexconv is included in the NITRO-SampleTools package. However, because ntexconv is used in a number of demos in the NITRO-SDK to create banner files (see makebanner), we have also included the ntexconv executable file (version 07/01/2005), in the NITRO-SDK package. If you require a newer version of ntexconv or newer source files, get them from NITRO-SampleTools, which is supplied separately.

How to Use

For information on using ntexconv, see the ntexconv manual.

Location

$NitroSDK/tools/bin/ntexconv.exe

See Also

makebanner, ntexconv manual, NITRO-SampleTools package
