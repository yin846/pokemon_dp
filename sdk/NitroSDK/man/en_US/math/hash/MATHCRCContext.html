MATHCRC*Context

C Specification

#include <nitro/math/crc.h>

typedef u8  MATHCRC8Context;
typedef u16 MATHCRC16Context;
typedef u32 MATHCRC32Context;

Description

This is the context structure used for CRC calculations. Initialize the structure with MATH_CRC*Init, then use MATH_CRC*Update to provide input data and MATH_CRC*GetHash to obtain the message digest value.
CRC (Cyclic Redundancy Check: an error detection method that uses check values.

See Also

MATH_Checksum*Init, MATH_Checksum*Update, MATH_Checksum*GetHash
