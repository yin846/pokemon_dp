SVC_Div

C Specification

#include <nitro/os/common/systemCall.h>
s32 SVC_Div( s32 numer, s32 demon );

Arguments

numer Numerator of fraction. Represents dividend of division here.
denom Denominator of fraction. Represents divisor of division here.

Return Values

Returns the result of the division operation.

Description

This function performs the calculation numer / denom and returns the result At the assembler level, it is restored from this function as follows:

r0 = numer / denom ,
r1 = numer % denom ,
r3 = | numer / denom | . Because the code size is compressed, it is not high speed.

See Also

CP_SetDiv*(), SVC_DivRem
