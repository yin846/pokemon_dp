SVC_GetCRC16

C Specification

#include <nitro/os/common/systemCall.h>
u16 SVC_GetCRC16( u32 start, const void *datap, u32 size );

Arguments

start Initial Value
datap Data Address
size Size (in Bytes)

Return Values

This is the calculated CRC-16 value.

Description

This calculates CRC-16. The data address datap needs to be a 2-byte boundary. The size size needs to be a multiple of 2. CRC stands for Cyclic Redundancy Check (Cyclic Redundancy Test) and is an error detection method.
