G3X_IsListRamOverflow

C Specification

#include <nitro/gx/g3x.h>

s32 G3X_IsListRamOverflow(void);

Arguments

None.

Return Values

No over flow generated (0) or overflow generated (not 0).

Description

This function returns a non-0 value when the number of polygons or vertices processed by the geometry engine is too great, and the polygon list RAM and vertex list RAM overflow. In this case the portion that overflowed will not be displayed. Returns a 0 value when there is no overflow. Once overflow occurs, it continues to return a non-0 value until G3X_ResetListRamOverflow is called.
