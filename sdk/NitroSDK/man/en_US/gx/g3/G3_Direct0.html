G3*_Direct0

C Specification

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_Direct0(int op);

// Below are APIs for creating the display list (command list):
void G3B_Direct0(GXDLInfo* info, int op);
void G3C_Direct0(GXDLInfo* info, int op);
void G3BS_Direct0(GXDLInfo* info, int op);
void G3CS_Direct0(GXDLInfo* info, int op);

Arguments

op Geometry command (G3OP_**** macro defined in g3.h)

Return Values

None.

Description

This function specifies the geometry command directly. A command with no arguments must be set in op.

See Also

G3_Direct1, G3_Direct2, G3_Direct3
