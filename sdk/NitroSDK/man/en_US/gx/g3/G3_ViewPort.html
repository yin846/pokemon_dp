G3*_ViewPort

C Specification

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_ViewPort(int x1, int y1, int x2, int y2);

// Below are APIs for creating the display list (command list)
void G3B_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3C_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3BS_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3CS_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);

Arguments

info Command List Information
x1   Lower left x-coordinate
y1   Lower left y-coordinate
x2 Upper right x-coordinate
y2 Upper right y-coordinate

Return Values

None.

Description

This function sets the viewport. The coordinate values for viewport settings are 0 - 255 for the x-coordinate and 0 - 191 for the y-coordinate.
