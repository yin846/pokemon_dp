G3X_SetFogTable

C Specification

#include <nitro/gx/g3x.h>

void G3X_SetFogTable(const u32 * fogTable);

Arguments

fogTable Pointer to the beginning of the fog density array set in the fog density table

Return Values

None.

Description

This function sets the fog density table. Data from fogTable[0] to fogTable[7] is transferred to the fog density table.

See Also

G3X_SetFog, G3X_SetFogColor
