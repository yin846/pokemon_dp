G3X_InitTable

C Specification

#include <nitro/gx/g3x.h>

void G3X_InitTable();

Arguments

None.

Return Values

None.

Description

This function clears the tables used during 3D rendering. Specifically, it does the following:

  • Zero-clears the edge color table
  • Zero-clears the fog table
  • Zero-clears the toon table
  • Zero-clears the shininess table
