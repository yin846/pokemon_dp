G3*_LoadMtx44

C Specification

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_LoadMtx44(cosnt G3Mtx44* m);

// Below are APIs for creating the display list (command list)
void G3B_LoadMtx44(GXDLInfo* info, const MtxFx44* m);
void G3C_LoadMtx44(GXDLInfo* info, const MtxFx44* m);
void G3BS_LoadMtx44(GXDLInfo* info, const MtxFx44* m);
void G3CS_LoadMtx44(GXDLInfo* info, const MtxFx44* m);

Arguments

info Command List Information
m   The pointer to a 4x4 matrix.

Return Values

None.

Description

This function sets a 4x4 matrix as the current matrix. Each element of the matrix is a signed fixed-point number (sign + 19-bit integer + 12-bit decimal).
