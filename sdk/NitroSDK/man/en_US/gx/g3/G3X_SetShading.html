G3X_SetShading

C Specification

#include <nitro/gx/g3x.h>

void G3X_SetShading(GXShading shading);

Arguments

shading Selects either toon shading or highlight shading

Return Values

None.

Description

This function selects whether to use toon shading or highlight shading. It determines which will be used when GX_POLYGONMODE_TOON is selected as the polygon mode with G3_PolygonAttr.

GX_SHADING_TOON Toon shading
GX_SHADING_HIGHLIGHT Highlight shading

The following indicates the type definition for GXShading format.

typedef enum
{
GX_SHADING_TOON = 0,
GX_SHADING_HIGHLIGHT = 1
}
GXShading;
