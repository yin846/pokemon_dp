G3X_IsLineBufferUnderflow

C Specification

#include <nitro/gx/g3x.h>

s32 G3X_IsLineBufferUnderflow(void);

Arguments

None.

Return Values

A value of 0 is returned if the line overflow has not occurred; a nonzero value is returned if overflow has occurred.

Description

This function returns a nonzero value if rendering is not in time for display and the color buffer underflows (i.e., the rendering engine experiences a line overflow). After an overflow occurs, the function will continue to return a nonzero value until the G3X_ResetLineBufferUnderflow function is called.
