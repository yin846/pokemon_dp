G3X_SetHOffset

C Specification

#include <nitro/gx/g3x.h>

void G3X_SetHOffset(int hOffset);

Arguments

hOffset Horizontal offset

Return Values

None.

Description

This function sets the horizontal offset for the 3D screen (BG). For 2D display, use G2_SetBG0Offset.

See Also

G2_SetBG0Offset
