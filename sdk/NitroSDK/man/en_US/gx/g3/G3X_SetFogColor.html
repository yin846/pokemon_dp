G3X_SetFogColor

C Specification

#include <nitro/gx/g3x.h>

void G3X_SetFogColor(GXRgb rgb, int alpha);

Arguments

rgb   RGB components for fog color
alpha alpha component for fog color (0 ≤ alpha ≤ 31)

Return Values

None.

Description

This function sets the fog color.
