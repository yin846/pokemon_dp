G3X_ResetMtxStackOverflow

C Specification

#include <nitro/gx/g3x.h>

void G3X_ResetMtxStackOverflow(void);

Arguments

None.

Return Values

None.

Description

This function resets matrix stack overflow/underflow flag. Also indirectly called when G3X_Init, G3X_Reset, G3X_InitMtxStack, or G3X_ResetMtxStack is called.

See Also

G3X_Init, G3X_Reset, G3X_InitMtxStack, G3X_ResetMtxStack
