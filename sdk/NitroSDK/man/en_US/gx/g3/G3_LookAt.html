G3_LookAt

C Specification

#include <nitro/gx/g3_util.h>

void G3_LookAt(
const VecFx32 * camPos,
const VecFx32 * camUp,
const VecFx32 * target,
MtxFx43 * mtx
);

Arguments

camPos Camera Position Vector Pointer
camUp Camera Up Vector Pointer
target Camera Focal Point Pointer
mtx The pointer to a 4x3 matrix.

Return Values

None.

Description

This function sets the matrix mode to Position/Vector mode and sets the camera matrix as the current matrix. When mtx is not NULL, the camera matrix will also be set in *mtx.*There is no need to normalize *camUp. The matrix that is set is as shown below.
