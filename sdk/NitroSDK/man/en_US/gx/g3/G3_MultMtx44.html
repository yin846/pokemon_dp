Arguments

info Command List Information
m   Pointer to the matrix to be multiplied

Return Values

None.

Description

This function premultiplies the current matrix by a 4x4 matrix. Each element of the matrix is a signed fixed-point number (sign + 19-bit integer + 12-bit decimal). If the current matrix is C, then the new matrix, Cnew, is in the format shown below.
