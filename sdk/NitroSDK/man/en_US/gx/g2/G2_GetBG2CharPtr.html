G2_GetBG2CharPtr

C Specification

#include <nitro/gx/g2.h>

void *G2_GetBG2CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG2 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG2 character data is stored. The return values are as follows for character BGs:

HW_BG_VRAM + (Last value in the value name of GXBGCharOffset enum type specified by GX_SetBGCharOffset) + (Last value in the value name of GXBGCharBase enum type specified by G2_SetBG2ControlText)

Since there is no character data for bitmap BGs, NULL is returned.

See Also

G2_SetBG2ControlText, ... G2_SetBG2Priority, G2_BG2Mosaic, G2_GetBG2ScrPtr, G2_SetBG2Offset
