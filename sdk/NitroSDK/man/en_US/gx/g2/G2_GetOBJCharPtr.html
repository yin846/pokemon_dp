G2_GetOBJCharPtr

C Specification

#include <nitro/gx/g2.h>

void *G2_GetOBJCharPtr();

Arguments

None.

Return Values

The pointer to the region where the main 2D Engine's OBJ character data is stored.

Description

This function only returns (void*)HW_OBJ_VRAM.
