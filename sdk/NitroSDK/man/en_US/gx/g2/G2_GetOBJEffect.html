G2_GetOBJEffect

C Specification

#include <nitro/gx/g2_oam.h>

GXOamEffect G2_GetOBJEffect(const GXOamAttr * oam);

Arguments

oam Pointer to the location storing the OAM data

Return Values

Effect type

Description

*Gets flip or affine transformation effects from *oam. For type GXOamEffect, refer to G2_SetOBJAttr.

See Also

G2_SetOBJAttr, G2_SetOBJEffect
