G2S_BG1Mosaic

C Specification

#include <nitro/gx/gx_bgcnt.h>

void G2S_BG1Mosaic(BOOL enable);

Arguments

enable Mosaic setting

Return Values

None.

Description

This function sets whether to apply mosaic to the sub 2D Engine's BG1 (for 2D display). When set to FALSE, mosaic is disabled. When set to anything else, mosaic is enabled.

See Also

G2S_SetBG1Control, G2S_SetBG1Priority, G2S_GetBG1ScrPtr, G2S_GetBG1CharPtr, G2S_SetBG1Offset
