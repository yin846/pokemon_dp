G2_GetBG0CharPtr

C Specification

#include <nitro/gx/g2.h>

void *G2_GetBG0CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG0 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG0 character data is stored. The return values are as follows:

HW_BG_VRAM + (Last value in the value name of GXBGCharOffset enum type specified by GX_SetBGCharOffset) + (Last value in the value name of GXBGCharBase enum type specified by G2_SetBG0Control) .

See Also

G2_SetBG0Control, G2_SetBG0Priority, G2_BG0Mosaic, G2_GetBG0ScrPtr, G2_SetBG0Offset
