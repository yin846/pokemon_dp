G2_GetBG1CharPtr

C Specification

#include <nitro/gx/g2.h>

void *G2_GetBG1CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG1 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG1 character data is stored. The return values are as follows:

HW_BG_VRAM + (Last value in the value name of GXBGCharOffset enum type specified by GX_SetBGCharOffset) + (Last value in the value name of GXBGCharBase enum type specified by G2_SetBG1Control) .

See Also

G2_SetBG1Control, G2_SetBG1Priority, G2_BG1Mosaic, G2_GetBG1ScrPtr, G2_SetBG1Offset
