G2S_GetBG0ScrPtr

C Specification

#include <nitro/gx/g2.h>

void *G2S_GetBG0ScrPtr();

Arguments

None.

Return Values

The pointer to the region storing the BG0 screen.

Description

This function returns a pointer to the region where sub 2D engine BG0 screen data is stored. The return values are as follows:

The value at the end of the enumerator type value name GXBGScrBase that is specified by HW_DB_BG_VRAM + (G2S_SetBG0Control).

See Also

G2S_SetBG0Control, G2S_SetBG0Priority, G2S_BG0Mosaic, G2S_GetBG0CharPtr, G2S_SetBG0Offset
