G2_SetOBJPosition

C Specification

#include <nitro/gx/g2_oam.h>

void G2_SetOBJPosition(GXOamAttr * oam, int x, int y);

Arguments

oam Pointer to the location storing the OAM data
x x-coordinate
y y-coordinate

Return Values

None.

Description

*This function changes the x-y coordinates of the OAM data stored in oam.
