G2S_GetBG2CharPtr

C Specification

#include <nitro/gx/g2.h>

void *G2S_GetBG2CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG2 character data is stored.

Description

This function returns a pointer to the region where sub 2D engine BG2 character data is stored. The return values are as follows for character BGs:

The value at the end of the enumerator type value name GXBGCharBase that is specified by HW_DB_BG_VRAM + (G2S_SetBG2ControlText).

Since there is no character data for bitmap BGs, NULL is returned.

See Also

G2S_SetBG2ControlText, ..., G2S_SetBG2Priority, G2S_BG2Mosaic, G2S_GetBG2ScrPtr, G2S_SetBG2Offset
