G2S_SetBG2Priority

C Specification

#include <nitro/gx/gx_bgcnt.h>

void G2S_SetBG2Priority(int priority);

Arguments

priority Display priority for BG2 (0-3)

Return Values

None.

Description

This function sets the sub 2D Engine's BG2 display priority order.
