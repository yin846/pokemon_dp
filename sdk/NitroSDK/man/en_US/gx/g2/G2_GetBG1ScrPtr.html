G2_GetBG1ScrPtr

C Specification

#include <nitro/gx/g2.h>

void *G2_GetBG1ScrPtr();

Arguments

None.

Return Values

A pointer to the region where BG1 screen data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG1 screen data is stored. The return values are as follows:

HW_BG_VRAM + (Last value in the value name of GXBGScrOffset enum type specified by GX_SetBGScrOffset) + (Last value in the value name of GXBGScrBase enum type specified by G2_SetBG0Control) .

See Also

G2_SetBG1Control, G2_SetBG1Priority, G2_BG1Mosaic, G2_GetBG1CharPtr, G2_SetBG1Offset
