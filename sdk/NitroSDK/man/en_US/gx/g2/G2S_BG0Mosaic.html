G2S_BG0Mosaic

C Specification

#include <nitro/gx/gx_bgcnt.h>

void G2S_BG0Mosaic(BOOL enable);

Arguments

enable Mosaic setting

Return Values

None.

Description

This function sets whether to apply mosaic to the sub 2D Engine's BG0 (for 2D display). When set to FALSE, mosaic is disabled. When set to anything else, mosaic is enabled.

See Also

G2_SetBG0Control, G2_SetBG0Priority, G2_GetBG0ScrPtr, G2_GetBG0CharPtr, G2_SetBG0Offset
