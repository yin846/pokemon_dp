GXS_HBlankOBJProc

C Specification

#include <nitro/gx/gx.h>

void GXS_HBlankOBJProc(BOOL proc);

Arguments

blank Flag that sets whether the OBJ rendering process occurs during H-Blank

Return Values

None.

Description

When the argument is TRUE, the OBJ rendering process occurs only during display period and is not performed during the H-Blank period. In this case, the maximum OBJ display performance cannot be used. When the argument is FALSE, the OBJ rendering process is performed for all H-Line periods (including H-Blank periods).
