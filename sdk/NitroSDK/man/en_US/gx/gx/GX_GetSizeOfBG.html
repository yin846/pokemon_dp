GX_GetSizeOfBG

C Specification

#include <nitro/gx/gx_vramcnt.h>

u32 GX_GetSizeOfBG(void);

Arguments

None.

Return Values

Size of the main 2D Engine BG.

Description

This function returns the number of bytes of memory allocated to the main 2D Engine BG.
