GX_GetSizeOfSubOBJ

C Specification

#include <nitro/gx/gx_vramcnt.h>

u32 GX_GetSizeOfSubOBJ(void);

Arguments

None.

Return Values

Size of the sub 2D Engine OBJ.

Description

This function returns the number of bytes of memory allocated to the sub 2D Engine OBJ.
