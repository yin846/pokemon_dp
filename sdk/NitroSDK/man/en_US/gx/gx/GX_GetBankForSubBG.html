GX_GetBankForSubBG

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubBG GX_GetBankForSubBG(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the sub 2D Engine's BG.

Description

This function returns settings of the VRAM bank allocated to the sub 2D Engine BG. For information on the GXVRamSubBG format, see GX_SetBankForSubBG.

See Also

GX_SetBankForSubBG, GX_ResetBankForSubBG, GX_DisableBankForSubBG
