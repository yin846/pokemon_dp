GX_SetMasterBrightness

C Specification

#include <nitro/gx/gx.h>

void GX_SetMasterBrightness(int brightness);

Arguments

brightness A value for adjusting brightness (equal to or greater than -16, equal to or less than 16)

Return Values

None.

Description

This function adjusts brightness in image output. Integers from -16 to 16 can be specified. Positive numbers increase brightness; negative numbers decrease it.

See Also

G2_SetBlendBrightness, GX_GetMasterBrightness
