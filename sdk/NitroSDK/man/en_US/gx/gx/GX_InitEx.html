GX_InitEx

C Specification

#include <nitro/gx/gx.h>

void GX_InitEx(u32 dma_no);

Arguments

dma_no Specify the DMA number to use with GX

Return Values

None.

Description

This function initializes all graphics registers that are not 3D-related. Internally, GX_SetPower (GX_POWER_ALL) is called. The GX function uses DMA 3 by default, but you can specify the DMA number to use in the GX function. You can also specify GX_DMA_NOT_USE in the argument to not use the DMA.
Before calling this function, OS_InitLock or OS_Init must be called.

See Also

GX_Init, G3X_Init, G3X_Reset, G3X_InitTable
