GX_BeginLoadTexPltt

C Specification

#include <nitro/gx/gx_load.h>

void GX_BeginLoadTexPltt(void);

Arguments

None.

Return Values

None.

Description

This function should be called before transferring texture palette to the texture palette slot. Internally, it allocates to LCDC the banks that were allocated to the texture palette slot and prepares to DMA transfer the texture palette using GX_LoadTexPltt.

Note: VRAM banks must be allocated to the texture palette slot using GX_SetBankForTexPltt in advance.

See Also

GX_LoadTexPltt, GX_EndLoadTexPltt
