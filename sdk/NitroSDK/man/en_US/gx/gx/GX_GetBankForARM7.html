GX_GetBankForARM7

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamARM7 GX_GetBankForARM7(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the ARM7 memory space.

Description

This function returns the settings of the VRAM bank allocated to ARM7. Refer to GX_SetBankForARM7 for details on GXVRamARM7 format.

See Also

GX_SetBankForARM7, GX_ResetBankForARM7, GX_DisableBankForARM7
