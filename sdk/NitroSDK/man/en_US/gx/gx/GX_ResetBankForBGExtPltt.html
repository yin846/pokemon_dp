GX_ResetBankForBGExtPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamBGExtPltt GX_ResetBankForBGExtPltt(void);

Arguments

None.

Return Values

VRAM bank allocated to the main 2D Engine's BG extended palette.

Description

This function allocates to LCDC the VRAM bank that was allocated to the BG Extended palette. Refer to GX_SetBankForBGExtPltt for details on GXVRamBGExtPltt format.

See Also

GX_SetBankForBGExtPltt, GX_GetBankForBGExtPltt, GX_DisableBankForBGExtPltt
