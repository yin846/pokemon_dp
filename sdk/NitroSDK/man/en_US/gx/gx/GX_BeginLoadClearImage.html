GX_BeginLoadClearImage

C Specification

#include <nitro/gx/gx_load.h>

void GX_BeginLoadClearImage(void);

Arguments

None.

Return Values

None.

Description

This function should be called before transferring the clear image to the clear image slot. Internally, it allocates to LCDC the banks that were allocated to the clear image slot and prepares to DMA transfer the clear image using GX_LoadClearImageColor and GX_LoadClearImageDepth.

Note: VRAM banks must be allocated to the clear image slot using GX_SetBankForClearImage in advance.

See Also

GX_LoadClearImageColor, GX_LoadClearImageDepth, GX_EndLoadClearImage
