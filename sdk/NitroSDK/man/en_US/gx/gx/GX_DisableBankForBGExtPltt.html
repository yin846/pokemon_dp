GX_DisableBankForBGExtPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamBGExtPltt GX_DisableBankForBGExtPltt(void);

Arguments

None.

Return Values

VRAM bank allocated to the main 2D Engine's BG extended palette.

Description

This function disables the VRAM bank allocated to the Main 2D Engine BG Extended palette. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to GX_SetBankForBGExtPltt for details on GXVRamBGExtPltt format.

See Also

GX_SetBankForBGExtPltt, GX_GetBankForBGExtPltt, GX_ResetBankForBGExtPltt
