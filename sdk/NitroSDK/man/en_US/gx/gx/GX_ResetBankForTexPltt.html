GX_ResetBankForTexPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamTexPltt GX_ResetBankForTexPltt(void);

Arguments

None.

Return Values

The VRAM bank allocated to the texture palette slot.

Description

This function allocates to LCDC the VRAM bank that was allocated to the texture palette slot. For information on the GXVRamTexPltt format, see GX_SetBankForTexPltt.

See Also

GX_SetBankForTexPltt, GX_GetBankForTexPltt, GX_DisableBankForTexPltt
