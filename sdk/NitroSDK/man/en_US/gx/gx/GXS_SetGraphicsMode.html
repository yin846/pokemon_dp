GXS_SetGraphicsMode

C Specification

#include <nitro/gx/gx.h>

void GXS_SetGraphicsMode(GXBGMode bgMode);

Arguments

bgMode Selects BG mode.

Return Values

None.

Description

This function sets the sub 2D Engine display mode. The bgMode argument selects the BG mode. Be aware that GX_BGMODE_6 can be specified on the main 2D Engine, but it is prohibited on the sub 2D Engine.

GX_BGMODE_0 BG0=Text, BG1=Text, BG2=Text, BG3=Text
GX_BGMODE_1 BG0=Text, BG1=Text, BG2=Text, BG3=Affine
GX_BGMODE_2 BG0=Text, BG1=Text, BG2=Affine, BG3=Affine
GX_BGMODE_3 BG0=Text, BG1=Text, BG2=Text, BG3=Affine Extended BG
GX_BGMODE_4 BG0=Text, BG1=Text, BG2=Affine, BG3=Affine Extended BG
GX_BGMODE_5 BG0=Text, BG1=Text, BG2=Affine Extended BG, BG3=Affine Extended BG

The following is the type definition for the GXBGMode type.

typedef enum
{
GX_BGMODE_0 = 0,
GX_BGMODE_1 = 1,
GX_BGMODE_2 = 2,
GX_BGMODE_3 = 3,
GX_BGMODE_4 = 4,
GX_BGMODE_5 = 5,
GX_BGMODE_6 = 6
}
GXBGMode;
