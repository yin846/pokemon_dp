GX_ResetBankForSubBG

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubBG GX_ResetBankForSubBG(void);

Arguments

None.

Return Values

VRAM bank that is allocated to the sub 2D Engine's BG.

Description

This function allocates to LCDC the VRAM bank that was allocated to the sub 2D Engine's BG. For information on the GXVRamSubBG format, see GX_SetBankForSubBG.

See Also

GX_SetBankForSubBG, GX_GetBankForSubBG, GX_DisableBankForSubBG
