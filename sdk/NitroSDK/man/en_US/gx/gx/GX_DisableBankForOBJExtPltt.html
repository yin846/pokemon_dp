GX_DisableBankForOBJExtPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamOBJExtPltt GX_DisableBankForOBJExtPltt(void);

Arguments

None.

Return Values

The VRAM bank allocated to the main 2D Engine's OBJ extended palette.

Description

This function disables the VRAM bank allocated to the Main 2D Engine's OBJ Extended palette. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. See GX_SetBankForOBJExtPltt for details on GXVRamOBJExtPltt types.

See Also

GX_SetBankForOBJExtPltt, GX_GetBankForOBJExtPltt, GX_ResetBankForOBJExtPltt
