GX_GetPower

C Specification

#include <nitro/gx/gx.h>

GXPower GX_GetPower(void);

Arguments

None.

Return Values

Status of each type of power control.

Description

This function acquires the status of each type of power control. For information on the GXPower format, see GX_SetPower.
