GX_DisableBankForARM7

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamARM7 GX_DisableBankForARM7(void);

Arguments

None.

Return Values

VRAM bank allocated to the ARM7 memory space.

Description

This function disables the VRAM bank allocated to the ARM7 memory space. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to GX_SetBankForARM7 for details on GXVRamARM7 format.

See Also

GX_SetBankForARM7, GX_GetBankForARM7, GX_ResetBankForARM7
