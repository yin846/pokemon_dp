GX_DisableBankForBG

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamBG GX_DisableBankForBG(void);

Arguments

None.

Return Values

VRAM bank that is allocated to the main 2D Engine's BG.

Description

This function disables the VRAM bank allocated to the Main 2D Engine BG. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. See GX_SetBankForBG for details on GXVRamBG types.

See Also

GX_SetBankForBG, GX_GetBankForBG, GX_ResetBankForBG
