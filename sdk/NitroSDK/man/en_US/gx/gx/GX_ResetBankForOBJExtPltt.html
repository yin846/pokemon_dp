GX_ResetBankForOBJExtPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamOBJExtPltt GX_ResetBankForOBJExtPltt(void);

Arguments

None.

Return Values

The VRAM bank allocated to the main 2D Engine's OBJ extended palette.

Description

This function allocates to LCDC the VRAM bank that was allocated to the the main 2D Engine's OBJ extended palette. See GX_SetBankForOBJExtPltt for details on GXVRamOBJExtPltt types.

See Also

GX_SetBankForOBJExtPltt, GX_GetBankForOBJExtPltt, GX_DisableBankForOBJExtPltt
