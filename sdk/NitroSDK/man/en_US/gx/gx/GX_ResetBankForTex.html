GX_ResetBankForTex

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamTex GX_ResetBankForTex(void);

Arguments

None.

Return Values

The VRAM bank allocated to the texture image slot.

Description

This function allocates to LCDC the VRAM bank that was allocated to the texture image slot. For information on GXVRamTex format, see GX_SetBankForTex.

See Also

GX_SetBankForTex, GX_GetBankForTex, GX_DisableBankForTex
