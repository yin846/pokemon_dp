GX_DisableBankForSubBGExtPltt

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubBGExtPltt GX_DisableBankForSubBGExtPltt(void);

Arguments

None.

Return Values

VRAM bank allocated to the sub 2D Engine's BG Extended palette.

Description

This function disables the VRAM bank allocated to the sub 2D Engine BG Extended palette. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to GX_SetBankForSubBGExtPltt for details on GXVRamSubBGExtPltt format.

See Also

GX_SetBankForSubBGExtPltt, GX_GetBankForSubBGExtPltt, GX_ResetBankForSubBGExtPltt
