GX_DisableBankForSubBG

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubBG GX_DisableBankForSubBG(void);

Arguments

None.

Return Values

VRAM bank that is allocated to the sub 2D Engine's BG.

Description

This function disables the VRAM bank allocated to the sub 2D Engine BG. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. For information on the GXVRamSubBG format, see GX_SetBankForSubBG.

See Also

GX_SetBankForSubBG, GX_GetBankForSubBG, GX_ResetBankForSubBG
