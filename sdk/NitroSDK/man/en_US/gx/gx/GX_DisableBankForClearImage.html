GX_DisableBankForClearImage

C Specification

#include <nitro/gx/gx_vramcnt.h>

GXVRamClearImage GX_DisableBankForClearImage(void);

Arguments

None.

Return Values

The VRAM bank allocated to clear image.

Description

This function disables the VRAM bank allocated to clear image. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to GX_SetBankForClearImage for more information on GXVRamClearImage format.

See Also

GX_SetBankForClearImage, GX_GetBankForClearImage, GX_ResetBankForClearImage
