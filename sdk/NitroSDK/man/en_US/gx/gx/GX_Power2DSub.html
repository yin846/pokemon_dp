GX_Power2DSub

C Specification

#include <nitro/gx/gx.h>

void GX_Power2DSub(BOOL enable);

Arguments

enable When FALSE, the sub 2D graphics engine stops.

Return Values

None.

Description

This function controls the sub 2D graphics engine. The 2D graphics engine stops when the argument is FALSE; it operates for all other values. Electrical consumption can be controlled by stopping the engine.

See Also

GX_SetPower, GX_Power2D, GX_Power2DMain
