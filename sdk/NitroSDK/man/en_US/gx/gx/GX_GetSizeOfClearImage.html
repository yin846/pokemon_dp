GX_GetSizeOfClearImage

#include <nitro/gx/gx_vramcnt.h>

u32 GX_GetSizeOfClearImage(void);

Arguments

None.

Return Values

Size of Clear Image memory.

Description

This function returns the number of bytes of memory allocated for clear image memory.
