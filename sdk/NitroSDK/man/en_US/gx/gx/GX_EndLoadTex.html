GX_EndLoadTex

C Specification

#include <nitro/gx/gx_load.h>

void GX_EndLoadTex(void);

Arguments

None.

Return Values

None.

Description

This function is called after transferring the texture image to the texture image slot. Internally, it restores to the texture image slot the VRAM bank that was allocated to LCDC by GX_BeginLoadTex.

See Also

GX_BeginLoadTex, GX_EndLoadTex
