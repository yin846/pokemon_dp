WBT_SIZE_USER_DATA

Definitions

#include <nitro/wbt.h>

#define WBT_SIZE_USER_DATA 9

Description

The maximum size of the user-defined data that can be sent with the WBT_PutUserData function. Users can batch-send small-scale data in a range smaller than this size.
