OS_SetArgumentBuffer

C Specification

#include <nitro/os.h>
void OS_SetArgumentBuffer( const char* buffer );

Arguments

None.

buffer Pointer to the argument buffer to be set

Return Values

None.

Description

This function sets the argument buffer used by functions such as OS_GetArgc, OS_GetArgv, and OS_GetOpt.

Although the argument buffer region in the system is used by default (argument data is embedded by the buryarg tool), the region can be changed using this function.

Argument data can easily be converted from a character string by using OS_ConvertToArguments().

Does nothing when the FINALROM builds.

See Also

OS_GetArgv, OS_GetArgc, OS_GetOpt, buryarg tool,
OS_GetArgumentBuffer, OS_ConvertToArguments
