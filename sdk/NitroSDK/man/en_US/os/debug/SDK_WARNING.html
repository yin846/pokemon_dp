SDK_WARNING*

C Specification

#include <nitro/misc.h>
SDK_WARNING(exp, fmt, ...);

Arguments

exp The condition parameter.
fmt The display character string (format character string).

Return Values

None.

Description

This macro examines a condition, and outputs a character string if the condition is not fulfilled. Unlike SDK_ASSERT(), it does not force an application to terminate. In accordance with the format character string fmt, a variable number of arguments is allowed.

Because this macro function is a debugging function, it will not perform the output operation in the RELEASE and the FINALROM builds.

See Also
