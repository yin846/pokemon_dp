OS_Terminate

C Specification

#include <nitro/os.h>

void OS_Terminate( void );

Arguments

None.

Return Values

None.

Description

This function sets the CPU to HALT status and loops the program. There is no return from this function.

Prohibits interrupts and calls OS_Halt() internally.

See Also

OS_Panic, OS_Halt, OS_Exit
