OS_Halt

C Specification

#include <nitro/os.h>

void OS_Halt( void );

Arguments

None.

Return Values

None.

Description

This function sets the CPU to the HALT state. If there is an interrupt to return from the HALT state, a return from the function occurs.

Internal Operation

HALT status comes about due to access to the CP15 register 7 of the system control coprocessor 15 with ARM9, and to a call of the system call with ARM7.

See Also

OS_Panic, OS_Terminate
