OS_EnableMainExArena

C Specification

#include <nitro/os.h>
void OS_EnableMainExArena( void );

Arguments

None.

Return Values

None.

Description

Declaration enabling use of expanded arena in main memory.

The main memory expanded arena can never be used when there is no expanded main memory. This applies to operations on the DS.

If you are going to call this function, do so before calling OS_InitArena or OS_Init.( OS_InitArena is called from OS_Init.))

To maintain compatibility with previous versions, there is also a way to specify NITRO_MEMSIZE=8M when compiling the OS library. (But even in this case, the main memory expanded arena cannot be used on the Nintendo DS.))

See Also

OS_InitArena, OS_DisableMainExArena
