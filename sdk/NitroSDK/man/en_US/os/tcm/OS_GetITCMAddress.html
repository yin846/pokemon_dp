OS_GetITCMAddress

C Specification

#include <nitro/os.h>
u32 OS_GetITCMAddress( void );

Arguments

None.

Return Values

The instruction TCM base address.

Description

Gets the instruction TCM base address This address is always 0x01FF8000 (constant HW_ITCM).
