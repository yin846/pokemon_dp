OS_EnableITCM

C Specification

#include <nitro/os.h>
void OS_EnableITCM( void );

Arguments

None.

Return Values

None.

Description

This function enables the Instruction TCM.

Internal Operation

Modifies register 1 of the system control co-processor CP15.
