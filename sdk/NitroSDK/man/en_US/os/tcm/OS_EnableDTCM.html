OS_EnableDTCM

C Specification

#include <nitro/os.h>
void OS_EnableDTCM( void );

Arguments

None.

Return Values

None.

Description

This function enables the Data TCM.

Internal Operation

Modifies register 1 of the system control co-processor CP15.
