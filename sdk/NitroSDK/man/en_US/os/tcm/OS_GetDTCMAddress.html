OS_GetDTCMAddress

C Specification

#include <nitro/os.h>
u32 OS_GetDTCMAddress( void );

Arguments

None.

Return Values

The data TCM base address.

Description

Gets the data TCM base address

Internal Operation

Modifies register 9 of the system control co-processor CP15.
