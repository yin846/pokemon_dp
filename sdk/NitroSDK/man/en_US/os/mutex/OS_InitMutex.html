OS_InitMutex

C Specification

#include <nitro/os.h>

void OS_InitMutex( OSMutex* mutex );

Arguments

mutex Pointer to Mutex structure to be initialized

Return Values

None.

Description

This function initializes Mutex. mutex is used to synchronize threads. Operations are unstable when a mutex in use is initialized.

See Also

OS_LockMutex, OS_UnlockMutex, OS_TryLockMutex
