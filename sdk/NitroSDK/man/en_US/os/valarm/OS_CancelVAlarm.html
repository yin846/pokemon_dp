OS_CancelVAlarm

C Specification

#include <nitro/os.h>
void OS_CancelVAlarm( OSVAlarm* alarm );

Arguments

alarm Pointer to the V-count alarm structure to cancel

Return Values

None.

Description

This function cancels a V-Count alarm. After it has been canceled, the V-Count alarm structure can be used for another V-Count alarm.

See Also

OS_InitVAlarm, OS_CancelVAlarms, OS_CancelAllVAlarms
