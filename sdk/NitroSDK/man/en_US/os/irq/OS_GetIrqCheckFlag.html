OS_GetIrqCheckFlag

C Specification

#include <nitro/os.h>

OSIrqMask OS_GetIrqCheckFlag( void );

Arguments

None.

Return Values

An IRQ interrupt check flag.

Description

Gets an IRQ check flag These flags are normally set by the IRQ interrupt handler. For information on the types of flags see OS_SetIrqCheckFlag.

See Also

OS_SetIrqCheckFlag, OS_ClearIrqCheckFlag

Internal Operation

In the ARM9, the predetermined location (DTCM_END - 8) in DTCM is designated to be the check flag region. This function operates on this region. In the ARM7, the predetermined location of the ARM7 WRAM (highest address - 8) is designated to be the check flag region. The function operates on this region.
