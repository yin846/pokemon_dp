OS_ClearIrqCheckFlag

C Specification

#include <nitro/os.h>

void OS_ClearIrqCheckFlag( OSIrqMask intr );

Arguments

intr Shows, as a logical sum, the IRQ interrupt cause that will clear the check flag.

Return Values

None.

Description

Clears IRQ interrupt check flags

These flags are normally set by the interrupt handler of each IRQ interrupt. For information on the types of flags, see OS_SetIrqCheckFlag.

See Also

OS_SetIrqCheckFlag, OS_GetIrqCheckFlag

Internal Operation

In the ARM9, the predetermined location (DTCM_END - 8) in DTCM is designated to be the check flag region. This function operates on this region. In the ARM7, the predetermined location of the ARM7 WRAM (highest address - 8) is designated to be the check flag region. The function operates on this region.
