List of Rumble Pak (VIB) API Functions

Function

VIB_Init Initializes the Rumble Pak library.
VIB_End Terminates the use of the Rumble Pak library.
VIB_SetCartridgePulloutCallback Registers the Rumble Pak removal callback.
VIB_StartPulse Starts pulse vibration.
VIB_StopPulse Stops pulse vibration.
VIB_IsCartridgeEnabled Checks whether a Rumble Pak is inserted.
VIB_IsExecuting Returns whether or not pulse vibration is currently being performed.

Structures

VIBPulseState Pulse vibration status

Constants

VIB_PULSE_NUM_MAX Gets the maximum number of pulses in a pulse set.
VIB_ON_TIME_MAX Gets the maximum value of vibration ON time per instance.
VIB_REST_TIME_MIN Gets the minimum vibration rest time.
