VIB_End

C Specification

#include <nitro/vib.h>
void VIB_End(void);

Arguments

None.

Return Values

None.

Description

Stops the use of the Rumble Pak library. The following processes are performed.

  • Pulse vibration is stopped
  • The pre-sleep callback registered by VIB_Init is deleted
  • The Game Pak removal callback is deleted
