Rumble Pak (overview)

Description

The DS Rumble Pak can be made to vibrate in any way based on control from software when using the NintendoDS option Game Pak. When using pulses, the type of pulse must be specified because ON/OFF of the pulse is required in order to make the DS Rumble Pak vibrate.

For pulse specifications, see VIB_StartPulse function and VIBPulseState function.

Reset and Sleep

When software is reset, pulse vibrations stop automatically.
When entering sleep mode, vibration is stopped by the pre-sleep callback registered within the init function.

Support Under IS-NITRO-EMULATOR

There are two possible problems when using a DS Rumble Pack under IS-NITRO-EMULATOR, depending on the version of IS-NITRO-EMULATOR.

  • The program cannot detect the Rumble Pack.
  • The Rumble Pack sometimes malfunctions when using printf.

This problem does not occur when using IS-NITRO-EMULATOR versions "F4," "G3," "H," or later. Versions with problems can be exchanged for a compatible version at no charge. Please contact the developer's help desk.
