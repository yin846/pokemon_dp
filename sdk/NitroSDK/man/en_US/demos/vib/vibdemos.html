Rumble Pak Sample Demo

Location

$NitroSDK/build/demos/vib

Compilation

Move to the $NitroSDK/build/demos/vib directory and enter

% make

 Executable (.nef, .srl) files will be generated in

$NitroSDK/build/demos/vib/(sample_name)/bin/(platform)/(target)/

  For example:

$NitroSDK/build/demos/vib/pulse_vib/bin/ARM9-TS/Release/

 

Execution

First, plug in the Rumble Pack used as an option Game Pak for the NintendoDS into the IS-NITRO-EMULATOR.

Move to the $NitroSDK/build/demos/vib/sample_name/bin/platform/target/ directory and perform one of the following:

  • Load and execute a *.nlf file with IS-NITRO-DEBUGGER.
  • Execute a *.srl file with IS-NITRO-DEBUGGER.

Description


pulse_vib Sample that allows any pulse to be generated for the Rumble Pak.

Operation

pulse_vib does not work properly with ensata.
