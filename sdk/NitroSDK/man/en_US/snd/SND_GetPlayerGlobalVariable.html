SND_GetPlayerGlobalVariable

C Specification

#include <nitro/snd.h>
s16 SND_GetPlayerGlobalVariable( int varNo );

Arguments

varNo The variable number. Takes values in the range of 0 to 15.

Return Values

The value of the Player global variable.

Description

This function gets the value of the sequence global variable.

The sequence global variable's initial value is -1. The variable is initialized when the system is initialized.

This function is carried out asynchronously with respect to ARM7 command functions. To acquire synchronization, you must use a function such as the SND_IsFinishedCommandTag function to confirm that the processing of ARM7 commands has completed.
