SND_GetPlayerStatus

C Specification

#include <nitro/snd.h>
u32 SND_GetPlayerStatus( void );

Arguments

None.

Return Values

Bit flags indicating whether each Player is active or not. The Players are represented in order, with the lowest bit representing Player 0. A Player is active if its representative bit is standing.

Description

Gets the status of the Players, showing whether they are active or not.

Caution

This function is carried out asynchronously with respect to ARM7 command functions. To acquire synchronization, you must use a function such as the SND_IsFinishedCommandTag function to confirm that the processing of ARM7 commands has completed.
