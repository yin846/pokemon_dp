CP_WaitDiv

C Specification

#include <nitro/cp.h>

void CP_WaitDiv( void );

Arguments

None.

Return Values

None.

Description

This function waits until the Divider is in a ready state.

Internal Operation

Reference DIVCNT(0x4000280) of the IO register.

See Also

CP_SetDivImm*, CP_SetDiv*, CP_IsDivBusy, CP_GetDivResult*, CP_GetDivRemainder*
