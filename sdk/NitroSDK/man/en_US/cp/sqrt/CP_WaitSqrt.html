CP_WaitSqrt

C Specification

#include <nitro/cp.h>

void CP_WaitSqrt( void );

Arguments

None.

Return Values

None.

Description

This function waits until the Square Root Calculator is in a ready state.

Internal Operation

Reference SQRTCNT(0x40002B0) of the IO register.

See Also

CP_SetSqrtImm*, CP_SetSqrt*, CP_IsSqrtBusy, CP_GetSqrtResult*
