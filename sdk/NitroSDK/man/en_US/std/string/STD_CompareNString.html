STD_CompareNString

C Specification

#include <nitro/std.h>

int STD_CompareNString( const char* str1, const char* str2, int len );
int STD_StrNCmp( const char* str1, const char* str2, int len );

Arguments

str1 Specifies the pointer to the compared string.
str2 Specifies the pointer to the compared string.
len Specifies the number of characters to compare.

Return Values

Returns 0 if the compared strings are the same.

Description

Compares only the initial len character in charstring str1 and charstring str2. The C standard function carries out the same operation as strncmp().

STD_StrNCmp() is defined as another name for STD_StrCompareNString().

See Also

STD_CompareString, STD_CompareLString
