Standard Library (STD) API Function List

Charstring Functions

STD_CopyString
(STD_StrCpy)		 Copies a string.
STD_CopyLString
(STD_StrLCpy)		 Copies only the specified length of a string.
STD_SearchString
(STD_StrStr)		 Searches for a specified string.
STD_GetStringLength
(STD_StrLen)		 Finds the length of a string.
STD_ConcatenateString
(STD_StrCat)		 Concantenates strings.
STD_CompareString
(STD_StrCmp)		 Compares strings.
STD_CompareNString
(STD_StrNCmp)		 Compares only a specified length of strings.
STD_CompareLString
(STD_StrLCmp)		 Compares strings.
STD_TSScanf Performs a form scan of a string in response to a variable argument.
STD_TVSScanf Performs a form scan of a string in response to an argument list.
