RTCInterrupt

Definitions

#include <nitro/rtc.h>
typedef void ( *RTCInterrupt )( void );

Arguments

None.

Return Values

None.

Description

This is the format for the user callback function that is called when an alarm interrupt is generated in the real time clock.
