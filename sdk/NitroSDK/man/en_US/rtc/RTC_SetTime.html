RTC_SetTime

C Specification

#include <nitro/rtc.h>
RTCResult RTC_SetTime( const RTCTime* time );

Arguments

time Pointer to RTCTime containing time information to set.

Return Values

If the request succeeds, it returns 0. If the request fails, it returns some numeric value other than 0.

Description

This function writes time data to the real time clock. It sends time data to ARM7 and returns the process result once it is received. This is a synchronous version of RTC_SetTimeAsync.

Caution

Do not define parameter values of nonexistent times.
This function requires a PXI interrupt for synchronization. Do not call this function when PXI interrupts are prohibited.
Because applications cannot change the date, use this API only for debugging during development.
