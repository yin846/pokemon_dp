RTCAlarmChan

Definitions

#include <nitro/rtc.h>
typedef enum
{ 
	RTC_ALARM_CHAN_1 = 0 ,
	RTC_ALARM_CHAN_2 ,
	RTC_ALARM_CHAN_MAX

} RTCAlarmChan;

Elements

RTC_ALARM_CHAN_1 Alarm Channel #1.
RTC_ALARM_CHAN_2 Alarm Channel #2.

Description

There are two alarm channels that you can register in the real time clock. Although each channel can operate independently, there is only one interrupt output common to both. Therefore, if interrupts are simultaneously generated in two channels, only one interrupt will be generated.
