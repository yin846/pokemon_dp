RTC_GetDateTime

C Specification

#include <nitro/rtc.h>
RTCResult RTC_GetDateTime( RTCDate* date ,RTCTime* time );

Arguments

date Pointer to RTCDate, where date information will be stored.
time Pointer to RTCTime, where time information will be stored.

Return Values

If the request succeeds, it returns 0. If the request fails, it returns some numeric value other than 0.

Description

This function reads date and time data from the real time clock. It queries ARM7 for date and time data, stores the information to caller-specified storage, and returns. This is a synchronous version of RTC_GetDateTimeAsync.

Caution

This function requires a PXI interrupt for synchronization. Do not call this function when PXI interrupts are prohibited.

See Also
