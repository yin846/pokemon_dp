VEC_Fx16Add

C Specification

#include <nitro/fx/fx_vec.h>

void VEC_Fx16Add(
const VecFx16 * a,
const VecFx16 * b,
VecFx16 * ab
);

Arguments

a The pointer to a vector.
b   The pointer to a vector.
ab   The sum of vector *a and vector *b

Return Values

None.

Description

This function performs addition of VecFx16 type vectors. a, b, and ab can be pointers to the same vector.

See Also

VecFx16, VEC_Fx16Subtract
