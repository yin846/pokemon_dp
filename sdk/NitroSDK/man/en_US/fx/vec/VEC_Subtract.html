VEC_Subtract

C Specification

#include <nitro/fx/fx_vec.h>

void VEC_Subtract(
const VecFx32 * a,
const VecFx32 * b,
VecFx32 * ab
);

Arguments

a The pointer to a vector.
b   The pointer to a vector.
ab   The difference of vector *a and vector *b.

Return Values

None.

Description

This function subtracts VecFx32 type vectors. a, b, and ab can be pointers to the same vector.

See Also

VecFx32, VEC_Add
