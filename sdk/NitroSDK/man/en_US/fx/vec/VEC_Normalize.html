VEC_Normalize

C Specification

#include <nitro/fx/fx_vec.h>

void VEC_Normalize(const VecFx32 * pSrc, VecFx32 * pDst);

Arguments

pSrc The pointer to a vector.
pDst   The pointer to the normalized vector.

Return Values

None.

Description

This function normalizes VecFx32 type vectors. The result is stored in *pDst. It uses the Divider and Square Root Calculator. pSrc and pDst can be pointers to the same vector.

Note: Correct calculation results cannot be obtaned using (pSrc.x×pSrc.x)+(pSrc.y×pSrc.y)+(pSrc.z×pSrc.z) > 0x100000000000000.
