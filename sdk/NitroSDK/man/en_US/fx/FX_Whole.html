FX_Whole

C Specification

#include <nitro/fx/fx.h>

s32 FX_Whole(fx32 v);

Arguments

v fx32-format fixed-point decimal.

Return Values

s32 type integer.

Description

This function returns the integer portion of an fx32 fixed-point decimal as an integer.
