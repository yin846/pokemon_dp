MTX_Scale22

C Specification

#include <nitro/fx/fx_mtx22.h>

void MTX_Scale22(MtxFx22 * pDst, fx32 x, fx32 y);

Arguments

pDst   The pointer to a 2x2 matrix.
x The scale factor along the x-axis.
y The scale factor along the y-axis.

Return Values

None.

Description

*This function sets the scale matrix in *pDst.
