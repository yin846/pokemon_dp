MTX_Copy44To33

C Specification

#include <nitro/fx/fx_mtx44.h>

void MTX_Copy44To33(const MtxFx44 * pSrc, MtxFx33 * pDst);

Arguments

pSrc The pointer to a 4x4 matrix.
pDst   The pointer to a 3x3 matrix.

Return Values

None.

Description

This function copies a 4x4 matrix to a 3x3 matrix. The extra elements of the 4x4 are ignored.

See Also

MtxFx44, MtxFx33
