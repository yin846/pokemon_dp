MTX_Transpose22

C Specification

#include <nitro/fx/fx_mtx22.h>

void MTX_Transpose22(const MtxFx22 * pSrc, MtxFx22 * pDst);

Arguments

pSrc The pointer to a 2x2 matrix.
pDst   The pointer to a 2x2 matrix.

Return Values

None.

Description

*This function sets the transpose matrix of *pSrc in *pDst. pSrc and pDst can be pointers to the same matrix.
