MTX_ScaleApply44

C Specification

#include <nitro/fx/fx_mtx44.h>

void MTX_ScaleApply44(
const MtxFx44 * pSrc,
MtxFx44 * pDst,
fx32 x,
fx32 y,
fx32 z
);

Arguments

pSrc The pointer to a 4x4 matrix.
pDst   The pointer to a 4x4 matrix.
x The scale factor along the x-axis.
y The scale factor along the y-axis.
z The scale factor along the z-axis.

Return Values

None.

Description

*This function premultiplies pSrc by the scale matrix and stores the result in pDst. pSrc and pDst can be pointers to the same matrix.

See Also

MtxFx44, fx32
