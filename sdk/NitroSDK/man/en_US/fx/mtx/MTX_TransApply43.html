MTX_TransApply43

C Specification

#include <nitro/fx/fx_mtx43.h>

void MTX_TransApply43(
const MtxFx43 * pSrc,
MtxFx43* pDst,
fx32 x,
fx32 y,
fx32 z
);

Arguments

pSrc The pointer to a 4x3 matrix.
pDst   The pointer to a 4x3 matrix.
x The component of translation parallel to the x axis.
y The component of translation parallel to the y axis.
z The component of translation parallel to the z axis.

Return Values

None.

Description

*This function premultiplies *pSrc by the parallel translation matrix and stores the result in *pDst. pSrc and pDst can be pointers to the same matrix.

See Also

MtxFx43, fx32
