MTX_Copy43To44

C Specification

#include <nitro/fx/fx_mtx43.h>

void MTX_Copy43To44(const MtxFx43 * pSrc, MtxFx44 * pDst);

Arguments

pSrc The pointer to a 4x3 matrix.
pDst   The pointer to a 4x4 matrix.

Return Values

None.

Description

This function copies a 4x3 matrix to a 4x4 matrix. The matrix element in the 4th column of the 4th row is FX32_ONE; the rest of the elements of the 4th column are set to 0.

See Also

MtxFx43, MtxFx44
