MTX_RotZ44

C Specification

#include <nitro/fx/fx_mtx44.h>

void MTX_RotZ44(MtxFx44 * pDst, fx32 sinVal, fx32 cosVal);

Arguments

pDst   The pointer to a 4x4 matrix.
sinVal   Sine of the angle of rotation
cosVal   Cosine of the angle of rotation

Return Values

None.

Description

*This function sets the matrix for rotation about the z-axis in **pDst.

See Also

MTX_RotX44, MTX_RotY44, MtxFx44, fx32
