FX_GetDivResultFx64c

C Specification

#include <nitro/fx/fx_cp.h>

fx64c FX_GetDivResultFx64c(void);

Arguments

None.

Return Values

Division results.

Description

This function returns the results of asynchronous fixed-point decimal division performed with FX_DivAsync and FX_DivAsyncImm in fx64c format.

See Also

FX_DivAsync, FX_DivAsyncImm, FX_GetDivResult
