FX_Mul32x64c

C Specification

#include <nitro/fx/fx.h>

fx32 FX_Mul32x64c(fx32 v32, fx64c v64c);

Arguments

v32 An fx32 format fixed-point decimal
v64 An fx64c format fixed-point decimal

Return Values

Result of multiplying v32 and v64.

Description

This function performs multiplication calculations of fx32 and fx64c types. It returns the result as an fx32 type. Values that are less than 1/4096 are rounded off. The section that contains this function was developed as a function to be called for THUMB code and is processed as a function that is declared as an inline function for the ARM code.
