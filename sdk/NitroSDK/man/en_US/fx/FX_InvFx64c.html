FX_InvFx64c

C Specification

#include <nitro/fx/fx_cp.h>

fx64c FX_InvFx64c(fx32 denom);

Arguments

denom fx32 format fixed-point decimal

Return Values

Reciprocal of denom.

Description

This function obtains an fx32-type reciprocal number. It returns the result as an fx64c type. The divider is used internally.

See Also

FX_Inv, FX_Mul32x64c, FX_DivFx64c
