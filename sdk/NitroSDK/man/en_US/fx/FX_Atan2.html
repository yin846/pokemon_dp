FX_Atan2

C Specification

#include <nitro/fx/fx_trig.h>

fx16 FX_Atan2( fx32 y, fx32 x );

Arguments

y An fx32 format fixed-point decimal.
x An fx32 format fixed-point decimal.

Return Values

Arctangent of y/x

Description

This finds the arctangent of y/x. The result that it returns is an fx16 type in radian units. Internally, a table is referenced.

See Also

FX_Atan, FX_AtanIdx, FX_Atan2Idx
