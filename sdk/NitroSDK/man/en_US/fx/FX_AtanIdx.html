FX_AtanIdx

C Specification

#include <nitro/fx/fx_trig.h>

u16 FX_AtanIdx( fx32   x );

Arguments

x An fx32 format fixed-point decimal

Return Values

Arctangent of x

Description

This determines the arctangent of x. The results are returned in a u16 format. The units of the result are equal to 1/65536 of a circle (2 π radians). (Example: A right angle would be 16384.) Internally, a table is referenced.

See Also

FX_Atan, FX_Atan2, FX_Atan2Idx
