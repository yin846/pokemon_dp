CARD_THREAD_PRIORITY_DEFAULT

Definition

#include <nitro/card.h>

#define CARD_THREAD_PRIORITY_DEFAULT 4

Description

The constant shows the initial priority value of the CARD asynchronous process thread. To change the priority of this thread, use the CARD_SetThreadPriority function. For details on the CARD asynchronous process thread, see the explanation of the CARD_GetThreadPriority function.

See Also

CARD_GetThreadPriority, CARD_SetThreadPriority
