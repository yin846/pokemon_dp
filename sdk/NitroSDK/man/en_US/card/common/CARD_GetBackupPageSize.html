CARD_GetBackupPageSize

C Specification

#include <nitro/card.h>

u32 CARD_GetBackupPageSize(void);

Arguments

None.

Return Values

Returns the page size of the backup device.

Description

Gets the page size of the backup device. The obtained value follows the specification made in the CARD_IdentifyBackup function.

See Also

CARD_IdentifyBackup, CARD_GetBackupTotalSize, CARD_GetBackupSectorSize
