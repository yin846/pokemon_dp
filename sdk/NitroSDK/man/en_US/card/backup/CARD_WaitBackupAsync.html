CARD_WaitBackupAsync

C Specification

#include <nitro/card.h>

BOOL CARD_WaitBackupAsync(void);

Arguments

None.

Return Values

The function returns a value of TRUE if the result of the last executed process is CARD_RESULT_SUCCESS. Otherwise, it returns a value of FALSE.

Description

The CARD backup device will remain blocked until the asynchronous process is complete. If an asynchronous process is not running, CARD_WaitRomAsync returns control immediately. If the result of the process is CARD_RESULT_SUCCESS, this function returns a value of TRUE. Otherwise, this function returns a value of FALSE. To obtain the detailed result value, call the CARD_GetResultCode function.

See Also

CARD_GetResultCode, CARD_TryWaitBackupAsync
