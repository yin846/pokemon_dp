CARD_ReadRomID

C Specification

#include <nitro/card.h>

u32 CARD_ReadRomID(void);

Arguments

None

Return Values

Returns the CARD-ID.

Description

Reads the CARD-ID from a CARD ROM device. A CARD-ID is a u32 value that contains information such as the manufacturer's code for the NITRO-CARD media.
